### 3.3 作业辅导标准化

**辅导流程**:
```
17:20-17:30 学生自主整理作业（记录作业清单）
17:30-18:00 自主完成作业（老师巡视，不直接给答案）

辅导原则:
├── 先独立思考（至少5分钟）
├── 再小组讨论（同桌互助）
├── 最后老师讲解（典型问题集中讲）
└── 错题记录（错题本，周末复习）

禁止行为:
✗ 直接告诉答案
✗ 代替学生完成作业
✗ 使用搜题软件
```

### 3.4 家长服务体系

**每日反馈**:
```
家长群每日推送（20:00前）:
📸 今日照片（3-5张，作业+机器人课+晚餐）
📝 作业完成情况（已完成/未完成清单）
🤖 机器人课内容（今日学了什么）
🍱 晚餐菜单
⭐ 今日之星（表扬1-2位学生）
📋 明日提醒（要带的东西、注意事项）
```

**每周反馈**:
```
周报内容:
├── 本周作业完成质量评价
├── 机器人课学习进度
├── 行为表现（纪律、卫生、互助）
├── 下周课程预告
└── 需要家长配合的事项
```

**每月反馈**:
```
月报内容:
├── 月度学习总结
├── 机器人作品照片/视频
├── 成长档案（对比月初和月末）
├── 下月计划
└── 1对1沟通预约（有问题深度沟通）
```
