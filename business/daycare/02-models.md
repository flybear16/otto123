## 二、三种合作模式详解

### 2.1 模式A: 自营晚托班（重资产）

**模式概述**:
自己租场地、招老师、收学生，OTTO机器人作为核心特色。

**投入预算（首店）**:
| 项目 | 金额 | 说明 |
|------|------|------|
| 场地押金+首季租金 | ¥30,000-60,000 | 社区底商100-150㎡，¥6,000-12,000/月 |
| 装修布置 | ¥20,000-40,000 | 简装+功能分区（作业区、活动区、餐饮区） |
| 设备家具 | ¥15,000-25,000 | 桌椅、书架、监控、厨房设备 |
| OTTO机器人设备 | ¥15,000-25,000 | 20-30套（含备用） |
| 首批运营资金 | ¥20,000 | 人员工资、食材、营销 |
| **首店总投入** | **¥100,000-170,000** | - |

**人员配置**:
```
满员30人配置:
├── 店长1人（可兼教学）: 统筹+机器人课
├── 托管老师1-2人: 接送+作业辅导+生活照料
└── 厨师/配餐1人（可外包）: 晚餐制作

薪资结构:
• 店长: ¥6,000-8,000/月 + 利润分红
• 托管老师: ¥4,000-5,000/月
• 厨师: ¥3,500-4,500/月（或外包¥15/人/餐）
```

**服务流程**:
```
16:30 学校放学
    ↓
16:30-17:00 校门口接学生 → 步行/校车返回托管点
    ↓
17:00-17:20 点心时间（水果+牛奶）
    ↓
17:20-18:00 作业辅导（托管老师监督）
    ↓
18:00-19:00 【OTTO机器人课】（周二、周四）
    或 自由活动/阅读（周一、三、五）
    ↓
19:00-19:30 晚餐时间
    ↓
19:30-20:00 自由活动/家长接回
    ↓
20:00 最晚接回时间
```

**定价策略**:
| 产品 | 价格 | 包含内容 |
|------|------|----------|
| 普通晚托 | ¥980/月 | 接送+作业+晚餐（无机器人课） |
| OTTO特色晚托 | ¥1,380/月 | 接送+作业+晚餐+每周2次机器人课 |
| 单报机器人课 | ¥680/月 | 仅周二、周四机器人课（不含托管） |

**财务模型（满员30人，OTTO特色班）**:
```
月收入: 30人 × ¥1,380 = ¥41,400

月支出:
├── 场地租金: ¥10,000
├── 人员工资: ¥18,000（店长+2老师+厨师）
├── 食材成本: ¥6,000（¥10/人/餐×30人×20天）
├── 设备折旧: ¥2,000
├── 水电杂费: ¥2,000
└── 营销杂费: ¥1,000
月总支出: ¥39,000

月净利润: ¥41,400 - ¥39,000 = ¥2,400
净利润率: 5.8%

回本周期: ¥150,000 ÷ ¥2,400 ≈ 62个月（5年+）
```

**⚠️ 关键发现**: 单店自营模式回本周期太长，必须提高客单价或降低成本。

**优化方案**:
- 提高客单价至¥1,580（净利润率提升至15%）
- 或增加周末兴趣班（额外收入）
- 或缩减场地成本（与社区合作免费场地）

### 2.2 模式B: 植入合作（轻资产）★主推

**模式概述**:
不自己开晚托班，与现有晚托机构合作，输出OTTO课程和设备，按学生人数或课时收费。

**合作方式**:
```
┌─────────────────────────────────────────────────────┐
│                    植入合作模式                      │
├─────────────────────────────────────────────────────┤
│  晚托机构（甲方）          我方（乙方）              │
│  ────────────────          ────────────             │
│  • 提供场地、学生、基础托管服务                      │
│  • 负责招生（将OTTO作为卖点）                        │
│  • 负责学生日常管理                                  │
│                            • 提供OTTO设备和课程      │
│                            • 派遣老师授课（或培训甲方老师）│
│                            • 负责机器人课教学质量    │
├─────────────────────────────────────────────────────┤
│  收费模式（二选一）:                                  │
│  模式B1: 按学生收费（我方收¥300-500/月/人）          │
│  模式B2: 按课时收费（¥200-300/课时）                 │
└─────────────────────────────────────────────────────┘
```

**为什么主推此模式**:
1. **零场地投入**: 不用租房子、装修
2. **零招生压力**: 晚托机构已有学生，转化即可
3. **轻资产快启动**: 1-2周即可落地
4. **风险可控**: 合作不成可快速换机构
5. **可规模化**: 一个老师可跑多个点

**合作机构筛选标准**:
| 维度 | 要求 | 加分项 |
|------|------|--------|
| **规模** | 现有学生≥20人 | 连锁品牌、多校区 |
| **场地** | 有可用教室或活动空间 | 已有功能教室 |
| **口碑** | 运营≥1年，无重大投诉 | 家长群活跃度高 |
| **意愿** | 明确想增加特色课程 | 已有其他素质课合作 |
| **位置** | 小学周边1公里内 | 高端社区周边 |
| **决策** | 老板/负责人可直接决策 | 曾成功引入过新课程 |

**收费模式对比**:

| 模式 | 收费方式 | 适合场景 | 我方收入（20人班） |
|------|----------|----------|-------------------|
| **B1: 按学生月费** | ¥400/月/人 | 长期合作、学生稳定 | ¥8,000/月 |
| **B2: 按课时** | ¥250/课时 | 短期合作、灵活安排 | ¥2,000/周（2次课） |
| **B3: 混合模式** | 月费¥300+课时费¥150 | 深度绑定、共同成长 | ¥6,000+¥1,200/月 |

**推荐B1模式**: 收入稳定，与机构利益绑定深。

**财务模型（B1模式，合作3个机构点）**:
```
单点收入: 20人 × ¥400 = ¥8,000/月
单点成本:
├── 师资: ¥3,000（每周2次课，¥375/次）
├── 设备折旧: ¥800（20套设备，按2年折）
├── 交通: ¥500
└── 杂费: ¥200
单点成本合计: ¥4,500
单点净利润: ¥8,000 - ¥4,500 = ¥3,500

3个机构点月净利润: ¥10,500
年净利润: ¥126,000

启动投入: 60套设备 × ¥300 = ¥18,000
回本周期: 2个月
```

**⚠️ 关键成功因素**:
- 找到优质合作机构（学生多、配合度高）
- 确保课程质量（孩子喜欢、家长认可）
- 建立稳定师资（或培训机构老师）

### 2.3 模式C: 品牌加盟（中资产扩张）

**模式概述**:
自营1-2家样板店跑通模式后，向第三方输出品牌、课程、运营体系，收取加盟费和管理费。

**加盟体系设计**:
| 项目 | 内容 |
|------|------|
| **加盟费** | ¥50,000-80,000（一次性） |
| **保证金** | ¥20,000（可退） |
| **管理费** | 营业额5-8%/月 |
| **课程更新费** | ¥5,000/年 |
| **设备采购** | 必须从总部采购（保证质量） |

**总部支持**:
```
开业前:
├── 选址评估
├── 装修设计方案
├── 人员招聘培训
├── 开业营销方案
└── 首批设备供应

运营期:
├── 月度运营督导
├── 课程持续更新
├── 师资培训（线上+线下）
├── 招生物料支持
├── 家长社群运营指导
└── 区域保护（1公里内不开第二家）
```

**加盟方画像**:
- 有教育行业经验（幼儿园、培训机构）
- 有场地资源（社区底商、写字楼）
- 认可OTTO教育理念
- 投资能力¥150,000-250,000
- 愿意全身心投入运营

**⚠️ 前提条件**: 必须先有2-3家成功自营/合作样板，证明模式可行。
