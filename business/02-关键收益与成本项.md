# 关键收益与成本项

> 一堂五步法 · 第三步：商业模式
> 修炼水平：L3-L4（从粗糙账到完整账）
> 核心方法：单元模型和 ROI · 科学决策三角形·宽度

**及格线在 L3 与 L4 之间。** L3 有粗糙账但缺关键项，L4 有完整账（所有关键收益项 + 关键成本项无遗漏）。本文的目标是把账算完整。

---

## L3 常见错误：缺关键项

### 三大被忽略的成本

| 被忽略的成本 | 具体表现 | OTTO 对应 |
|-------------|----------|-----------|
| **经营成本** | 设备折旧、场地隐性成本、交通物流 | 器材损耗 15%/年、BD 拜访交通费 |
| **组织成本** | 招聘培训、管理沟通、团队协作 | 师资培养 ¥2,000-5,000/人、BD 培养 3-6 月 |
| **财务成本** | 资金占用、坏账风险、税务 | 学校首年 ¥85,000 亏损需缓冲、晚托机构流失 20%/年 |

> 教育行业最容易被忽略的成本是**师资培养和组织管理**——它不像器材那样有明确发票，但真实消耗大量资源。

## 完整成本项清单

### 按成本性质分类

```
完整成本结构：

固定成本（每月发生，不随学生数变化）：
├── 全职人力：¥10,000-15,000/人/月
├── 办公场地：¥10,000/月
├── 办公杂费：¥5,000/月（水电、网络、文具）
└── 管理费用：¥5,000/月（财务、法务、行政）

变动成本（随业务量变化）：
├── 兼职师资：¥375-750/课时
├── 器材成本：套件 ¥250/台、扩展模块 ¥120-300/个
├── 交通物流：¥500-1,000/月/BD
├── 体验课补贴：¥30/人（体验课免费，成本自担）
└── 营销投放：¥5,000-10,000/月（C 端渠道）

隐性成本（常被忽略）：
├── 师资招聘：猎头费/招聘平台费
├── 师资培训：¥2,000-5,000/人（2-4 周培训+4-8 周带教）
├── 师资流失：6 个月留存率 50%，流失后需重新招聘培训
├── 客户维护：BD 定期拜访、活动组织、关系维护
├── 设备损耗：年损耗率约 15%
└── 管理损耗：10 个点以上需加管理层级

财务成本：
├── 资金占用：学校首年亏损 ¥85,000 需缓冲
├── 坏账准备：晚托机构流失 20%/年
└── 税务成本：增值税+企业所得税（约 6-15%）
```

## 三种模式的完整账本对比

### 学校社团课（单校）

```
收入项（首年）：
├── 学期课程费：¥1,280 × 30人 = ¥38,400
├── 寒假集训：¥3,000 × 10人 = ¥30,000
├── 扩展模块销售：¥10,000
├── 校本课程定制：¥20,000（已含在投入中，首年不计）
└── 竞赛指导费：¥5,000（已含在投入中，首年不计）
    合计：¥78,400

成本项（首年）：
├── 器材投入：¥45,000（30套入门+5套备用+扩展）
├── 课程开发：¥25,000（定制+资料）
├── 师训服务：¥10,000（3人培训+认证）
├── 驻场支持：¥10,000（前4次课）
├── 竞赛指导：¥5,000
├── 兼职师资：¥48,000（32课时×¥1,500/课时）
├── 交通物流：¥3,000
└── 管理分摊：¥5,000
    合计：¥151,000

首年净贡献：¥78,400 - ¥151,000 = -¥72,600（亏损，建立标杆）

────────────────────────────────────────

收入项（成熟期·第2年起）：
├── 学期课程费：¥1,580 × 50人 × 2学期 = ¥158,000
├── 寒暑假集训：¥3,000 × 20人 = ¥60,000
├── 扩展模块：¥30,000
├── 竞赛服务：¥20,000
└── 新师训：¥6,000
    合计：¥274,000

成本项（成熟期·第2年起）：
├── 兼职师资：¥64,000（64课时×¥1,000/课时）
├── 器材补充：¥18,000
├── 运营支持：¥20,000
├── 竞赛指导：¥10,000
├── 交通物流：¥3,000
└── 管理分摊：¥5,000
    合计：¥120,000

成熟期净贡献：¥274,000 - ¥120,000 = ¥154,000（年利润）
毛利率：56%
```

### 晚托植入点（单点）

```
收入项（月度）：
├── 课程费：¥400 × 20人 = ¥8,000
├── 扩展模块：¥500（平均）
    合计：¥8,500/月

成本项（月度）：
├── 兼职师资：¥2,250（6课时×¥375/课时）
├── 器材折旧：¥300（月均）
├── 交通物流：¥400
├── 机构分成：¥0（课程费全归 OTTO）
└── 管理分摊：¥200
    合计：¥3,150/月

月净贡献：¥8,500 - ¥3,150 = ¥5,350
毛利率：63%

首期投入：¥3,300（基础设备）
回本周期：0.6 个月

────────────────────────────────────────

⚠️ 隐性成本提醒：
├── 晚托机构年流失率 20%（经营不稳定）
├── 每个流失点损失：¥3,300 投入 + 培养周期
└── 应对：同时运营 10+ 个点分散风险
```

### C 端周末班（单校区）

```
收入项（月度·满员）：
├── 课程费：¥1,380 × 60人 = ¥82,800
├── 扩展模块：¥3,000
├── 寒暑假集训：¥30,000/月（集中月）
    合计：¥85,800/月（常规月）

成本项（月度·满员）：
├── 场地租金：¥30,000
├── 全职师资：¥15,000 × 2人 = ¥30,000
├── 兼职师资：¥6,000
├── 器材折旧：¥2,000
├── 营销投放：¥8,000
├── 体验课补贴：¥3,000（100人×¥30）
├── 水电物业：¥3,000
└── 管理费用：¥5,000
    合计：¥87,000/月

月净贡献：¥85,800 - ¥87,000 = -¥1,200（满员仍微亏！）

────────────────────────────────────────

⚠️ 关键发现：周末班满员状态下月利润仅 -¥1,200
├── 核心问题：场地+人力固定成本过高（¥60,000/月）
├── 盈利路径：寒暑假集训 + 扩展模块复购 + 竞赛班升级
├── 集训月收入增量：+¥30,000 → 月利润 ¥28,800
└── 结论：周末班靠常规课程微亏，靠集训/竞赛盈利
```

## 关键成本项深度拆解

### 师资成本（最大变量）

```
师资成本全链路：

招聘阶段：
├── 招聘渠道费：¥500-2,000/人（平台/猎头）
├── 筛选通过率：30/100 人进入面试
└── 面试通过率：15/30 人进入培训

培训阶段：
├── 培训时长：2-4 周（课程体系+教学法+设备操作）
├── 培训成本：¥2,000-5,000/人（含带教老师时间）
├── 试讲通过率：10/15 人上岗
└── 上岗留存率：6/10 人留存 6 个月

运营阶段：
├── 兼职课时费：¥375-750/课时（按能力分级）
├── 全职月薪：¥8,000-15,000
├── 月度管理成本：¥500/人（排课+沟通+考核）
└── 6 个月留存率：约 50%（兼职）、80%（全职）

流失代价：
├── 直接损失：¥2,000-5,000 培训费 + 招聘费
├── 间接损失：学生适应新老师可能流失
└── 应对：师训体系标准化 + 晋升通道 + 兼职转全职
```

### 设备成本

```
设备成本全链路：

采购成本：
├── 入门套件：¥250/台（成本价）
├── 扩展模块：¥120-300/个（成本价）
├── 备用设备：每 15 台配 3 台（20% 冗余）
└── 采购议价空间：50+ 套时可降 10-20%

折旧与损耗：
├── 年折旧率：约 15%（3 年使用周期）
├── 人为损耗：约 5%/年（学生操作不当）
├── 维修成本：约 ¥50/台/年
└── 总持有成本：约 ¥70/台/年

升级成本：
├── 固件 OTA：¥0（远程升级）
├── 硬件升级：¥50-100/台（新模块）
└── 升级周期：每 1-2 年
```

### 获客成本

```
获客成本对比（CAC）：

学校社团课：
├── BD 拜访：¥500/校（交通+材料）
├── 方案制作：¥1,000/校
├── 体验课：¥0（学校提供场地和学生）
├── 签约周期：2-3 个月
├── 首校获客：¥1,500 / 30人 = ¥50/人
└── 后续扩班：¥0/人（已有渠道）

晚托植入：
├── BD 拜访：¥300/机构
├── 体验课：¥0（机构提供场地和学生）
├── 签约周期：2-4 周
├── 首点获客：¥300 / 20人 = ¥15/人
└── 后续扩班：¥0/人

C 端周末班：
├── 地推体验课：¥30/人（补贴）
├── 营销投放：¥5,000-10,000/月
├── 转化率：30%（体验→正式）
├── 有效 CAC：¥30 / 30% = ¥100/人（体验渠道）
├── 营销 CAC：¥8,000 / 40人 = ¥200/人（投放渠道）
└── 综合 CAC：¥500-800/人（含品牌建设分摊）
```

## L4 完整账检查清单

> 用这个清单确保你的账本没有遗漏：

- [ ] **固定成本**：人力、办公、杂费、管理——是否完整？
- [ ] **变动成本**：师资、器材、交通、营销——是否按量计算？
- [ ] **隐性成本**：招聘培训、设备损耗、客户维护——是否计入？
- [ ] **财务成本**：资金占用、坏账准备、税务——是否预留？
- [ ] **直接收益**：所有收费项目——是否完整列出？
- [ ] **间接收益**：品牌、转介绍、数据——是否有量化估算？
- [ ] **战略性收益**：师资资产、课程 IP、供应链——是否纳入长期考量？
- [ ] **时间维度**：首年 vs 成熟期——是否分别计算？
- [ ] **最坏情况**：核心假设不成立时——亏损多少？能承受吗？
