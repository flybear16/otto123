## 一、财务模型假设

| 假设项 | 数值 | 说明 |
|--------|------|------|
| **客单价（学期课程）** | 4000元/班 | 16次课，20-30人/班 |

```

├── 课程：高毛利、可持续、现金流好
```

## 四、敏感性分析

### 4.1 关键变量影响

| 变量 | 基准值 | 变化 | 对Year 3净利润影响 |
|------|--------|------|-------------------|
| **学校数量** | 25所 | +5所（+20%） | +11.2万（+20%）|
| | | -5所（-20%） | -11.2万（-20%）|
| **客单价** | 4000元/班 | +500元（+12.5%） | +11.2万（+20%）|
| | | -500元（-12.5%） | -11.2万（-20%）|
| **毛利率** | 60% | +5% | +14.5万（+26%）|
| | | -5% | -14.5万（-26%）|
| **续约率** | 70% | +10% | +5.6万（+10%）|
| | | -10% | -8.4万（-15%）|

### 4.2 盈亏平衡点

```
盈亏平衡分析（Year 3）：

固定成本：约90万/年（人员+运营）
变动成本率：约40%
毛利率：60%

盈亏平衡收入 = 固定成本 / 毛利率
            = 90万 / 0.6
            = 150万/年

对应学校数：约13所学校（每校平均11.5万收入）

结论：
├── Year 1：亏损正常（投入期）
├── Year 2：接近盈亏平衡
└── Year 3：大幅盈利（25所学校）
```

## 七、财务模型总结

### 7.1 核心结论

```
财务可行性：✅ 可行

关键数字：
├── 启动资金：25-35万
├── 盈亏平衡：Year 2末-Year 3初
├── 3年净利润：40.7万
├── 3年收入：416万
├── Year 3净利率：19.3%
└── 单校3年累计毛利：38.4万

关键假设：
├── 首年3所学校（深度合作）
├── 次年10所学校（验证模式）
├── 第3年25所学校（规模化）
├── 客单价4000元/班/学期
└── 毛利率60%
```

### 7.2 风险提示

| 风险 | 影响 | 应对 |
|------|------|------|
| 学校拓展慢于预期 | 亏损期延长 | 控制成本，单点深耕 |
| 价格竞争激烈 | 毛利率下降 | 强化差异化，不打价格战 |
| 师资供给不足 | 服务质量下降 | 建立师训体系，提高人效 |
| 政策变化 | 市场需求萎缩 | 关注政策，多元布局 |
