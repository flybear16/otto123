### 2.2 体验课设计（引流产品）

**目标**: 让家长在2小时内看到孩子的兴奋和潜力，产生付费意愿。

**课程流程**:
```
【0-15分钟】破冰与导入
├── 机器人舞蹈表演（吸引眼球）
├── 自我介绍+分组（4-6人/组）
└── 今日任务发布："创造你的第一个机器人"

【15-60分钟】动手组装
├── 发放OTTO套件（已预装部分组件）
├── 老师演示关键步骤（视频+现场）
├── 学生动手组装（老师巡回指导）
└── 完成基础组装，通电测试

【60-90分钟】AI语音交互
├── 连接WiFi，激活语音功能
├── 学习唤醒词和基础指令
├── 与机器人对话互动
└── 尝试语音控制动作

【90-110分钟】创意挑战
├── 小组任务：让机器人完成指定动作
├── 自由探索：尝试更多语音指令
└── 创意改造：给机器人起名字、设计简单装饰

【110-120分钟】展示与转化
├── 每组展示作品（1分钟）
├── 颁发"小小创客"证书
├── 家长进入，孩子向家长展示
├── 课程顾问介绍正式课，现场报名优惠
└── 扫码入群，领取学习资料
```

**转化策略**:
- 现场优惠：体验课当天报名立减¥200
- 早鸟价：前10名报名再减¥100
- 团购价：3人成团各减¥150
- 赠品：报名送OTTO主题T恤/贴纸

### 2.3 周末学期班（主力产品）

**课程设置**:
```
时间安排:
├── 周六上午班: 9:00-11:00
├── 周六下午班: 14:00-16:00
├── 周日上午班: 9:00-11:00
└── 周日下午班: 14:00-16:00

班级规模: 8-12人/班（小班教学）
学期周期: 16周（对应学校学期）
```

**课程大纲**:

| 阶段 | 周次 | 主题 | 核心内容 | 产出 |
|------|------|------|----------|------|
| **基础篇** | 1-2 | 认识OTTO | 机器人历史、安全规范、机械结构 | 课程契约 |
| | 3-4 | 机械组装 | 舵机原理、四足结构、稳定性调试 | 可站立机器人 |
| | 5-6 | 电路连接 | 主控板接口、电源管理、故障排查 | 通电自检通过 |
| **智能篇** | 7-8 | 语音唤醒 | 配网流程、唤醒词训练、首次对话 | 成功语音唤醒 |
| | 9-10 | 动作指令 | 基础动作、语音控制、动作组合 | 完成5个动作指令 |
| | 11-12 | 创意编程 | 图形化编程、事件触发、循环结构 | 自编动作程序 |
| **创造篇** | 13-14 | 外壳DIY | 3D打印、材料选择、个性化设计 | 专属外观机器人 |
| | 15 | 项目实战 | 小组项目、功能实现、测试迭代 | 完整项目作品 |
| | 16 | 成果展示 | 作品展示、竞赛评比、结业典礼 | 获奖证书 |

**增值服务**:
- 家长群每日反馈（照片+学习要点）
- 每月1次1对1学习报告
- 学期末作品视频剪辑
- 优秀学员推荐参加竞赛

### 2.4 假期集训营（深度产品）

**产品形态**:
```
【5天全日营】
时间: 寒暑假连续5天，每天9:00-17:00
包含: 午餐+上下午各一次课+项目实践
定价: ¥2,980-3,980（根据城市级别）
容量: 15-20人/期

【3天半日营】
时间: 连续3天，每天9:00-12:00 或 14:00-17:00
定价: ¥1,280-1,580
容量: 10-12人/期
```

**5天全日营安排**:
```
Day 1: 机器人基础
├── 上午: OTTO组装与调试
├── 下午: 电路原理与故障排查
└── 晚间: 机器人电影之夜

Day 2: AI语音交互
├── 上午: 语音识别原理与配置
├── 下午: 语音指令编程实践
└── 晚间: 创意工作坊（外壳设计）

Day 3: 动作编程进阶
├── 上午: 图形化编程深度应用
├── 下午: 复杂动作序列设计
└── 晚间: 小组项目选题

Day 4: 项目实战
├── 全天: 小组项目开发（老师巡回指导）
└── 晚间: 项目中期评审

Day 5: 成果展示
├── 上午: 项目完善与彩排
├── 下午: 家长开放日+成果展示
└── 结业典礼+颁奖
```

**特色亮点**:
- 高强度沉浸式学习
- 完整项目制（PBL）体验
- 跨学科融合（编程+设计+表达）
- 社交场景（团队合作、结交朋友）

### 2.5 竞赛冲刺班（高端产品）

**目标赛事**:
- 全国青少年科技创新大赛
- 全国中小学信息技术创新与实践大赛（NOC）
- 中国青少年机器人竞赛
- 各省市科技节/创客大赛

**课程安排**:
```
赛前8周:
Week 1-2: 竞赛规则解读+技术强化
Week 3-4: 项目选题与原型开发
Week 5-6: 项目完善与文档准备
Week 7-8: 模拟答辩与赛前冲刺

20课时分配:
├── 理论课: 6课时（规则、案例、策略）
├── 实践课: 10课时（项目开发）
├── 模拟赛: 3课时（全真模拟）
└── 答辩辅导: 1课时（演讲技巧）
```

**交付成果**:
- 竞赛作品1件（可运行、可展示）
- 技术文档1套（设计报告、使用说明）
- 演示PPT和视频
- 参赛报名全程协助
