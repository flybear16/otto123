## 五、财务模型

### 5.1 单校区财务模型（满员60人）

**收入预测**:
| 项目 | 单价 | 人数 | 月收入 | 备注 |
|------|------|------|--------|------|
| 周末学期班 | ¥1,380 | 40人 | ¥55,200 | 主力产品 |
| 假期集训营 | ¥3,280 | 10人 | ¥32,800 | 按季度折算 |
| 竞赛班 | ¥2,280 | 6人 | ¥13,680 | 按季度折算 |
| 体验课 | ¥29.9 | 20人 | ¥598 | 引流产品 |
| **月均收入** | - | - | **~¥75,000** | - |

**成本结构**:
| 项目 | 金额 | 占比 |
|------|------|------|
| 场地租金 | ¥12,000 | 16% |
| 人员工资 | ¥35,000 | 47% |
| 设备折旧 | ¥3,000 | 4% |
| 营销费用 | ¥7,500 | 10% |
| 水电杂费 | ¥3,000 | 4% |
| 其他 | ¥2,500 | 3% |
| **月总成本** | **¥63,000** | **84%** |

**利润**:
```
月净利润: ¥75,000 - ¥63,000 = ¥12,000
净利润率: 16%

年净利润: ¥144,000

回本周期（首店投入¥200,000）: 约17个月
```

### 5.2 盈利优化路径

**提升收入**:
1. 提高满班率（60→80人）
2. 增加高客单价产品（私教、竞赛班）
3. 开发周边产品（OTTO配件、教材）
4. 企业/学校B端服务

**降低成本**:
1. 优化人员结构（兼职老师、共享行政）
2. 降低场地成本（社区合作、学校场地）
3. 提高设备利用率（多班复用）

**优化后模型**:
```
满员80人，月均收入: ¥95,000
优化成本后月支出: ¥68,000
月净利润: ¥27,000
净利润率: 28%
年净利润: ¥324,000
回本周期: 8个月
```
