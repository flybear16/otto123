## 四、运营交付标准

### 4.1 校区运营标准

**场地要求**:
```
面积: 80-150㎡（根据班级数量）
分区:
├── 教学区: 60%（课桌椅、投影/电视）
├── 展示区: 20%（学生作品、奖杯证书）
├── 储物区: 15%（设备、物料）
└── 接待区: 5%（前台、等候区）

设施:
├── 网络: 稳定WiFi（供OTTO配网）
├── 电源: 充足插座（设备充电）
├── 安全: 监控全覆盖、消防器材
└── 氛围: 创客风格装饰、学生作品墙
```

**人员配置**:
```
基础配置（2个班）:
├── 校区负责人1人（兼教学）
├── 授课老师1-2人
├── 课程顾问1人（兼行政）
└── 兼职地推/助教（按需）

满负荷配置（6个班）:
├── 校区负责人1人
├── 授课老师3-4人
├── 课程顾问2人
├── 行政/客服1人
└── 市场专员1人
```

### 4.2 教学交付标准

**课前准备**:
```
提前1天:
├── 教案复习、教具准备
├── 设备充电、测试
├── 教室布置、卫生检查
└── 学员出勤确认

提前30分钟:
├── 到场、设备最终检查
├── 迎接学生、签到
└── 课前热身活动
```

**课中标准**:
```
时间分配:
├── 导入: 5-10分钟
├── 讲解演示: 15-20分钟
├── 动手实践: 40-50分钟
├── 分享展示: 10-15分钟
└── 总结作业: 5分钟

行为准则:
├── 每15分钟关注一次全体学生状态
├── 每节课至少与每个学生互动1次
├── 鼓励学生提问和表达
├── 及时拍照记录（用于课后反馈）
└── 安全第一（设备使用、行为规范）
```

**课后反馈**:
```
课后2小时内:
├── 家长群发送课堂照片（3-5张）
├── 发送学习要点和作业
└── 回答家长问题

每周:
├── 发送周学习报告
└── 与重点家长1对1沟通

每月:
├── 发送月学习报告
├── 组织家长开放日（可选）
└── 收集满意度调查
```

### 4.3 服务质量监控

**关键指标（KPI）**:
| 指标 | 目标值 | 监控频率 |
|------|--------|----------|
| 出勤率 | >90% | 每周 |
| 满意度 | >4.5/5 | 每月 |
| 续费率 | >70% | 每学期 |
| 转介绍率 | >30% | 每学期 |
| 体验课转化率 | >25% | 每月 |
| 投诉率 | <2% | 每月 |

**神秘顾客制度**:
- 每月安排1次神秘顾客体验
- 从预约、接待、试听、跟进全流程评估
- 出具评估报告，限期整改
