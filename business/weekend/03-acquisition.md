## 三、获客与转化策略

### 3.1 获客渠道矩阵

| 渠道 | 成本 | 效果 | 优先级 | 具体动作 |
|------|------|------|--------|----------|
| **转介绍** | 低 | 极高 | ⭐⭐⭐⭐⭐ | 老带新奖励、推荐有礼 |
| **学校合作** | 中 | 高 | ⭐⭐⭐⭐⭐ | 进校体验课、社团课引流 |
| **社区地推** | 中 | 中 | ⭐⭐⭐⭐ | 小学门口、社区活动、商场中庭 |
| **线上投放** | 高 | 中 | ⭐⭐⭐ | 抖音本地推、小红书种草 |
| **异业合作** | 低 | 中 | ⭐⭐⭐⭐ | 与书店、科技馆、游乐场合作 |
| **内容营销** | 低 | 长期 | ⭐⭐⭐ | 公众号、视频号、知乎 |

### 3.2 转介绍体系（核心渠道）

**转介绍机制**:
```
老学员推荐新学员:
├── 推荐1人: 奖励¥100现金券
├── 推荐2人: 奖励¥300现金券
├── 推荐3人: 奖励¥500现金券 + 赠送1次竞赛辅导
└── 推荐5人: 成为"OTTO推广大使"，享受全年9折

被推荐人福利:
└── 首次报名立减¥100
```

**转介绍话术**:
```
给家长的话术:
"张妈妈，感谢您选择OTTO 123！
如果您觉得我们的课程不错，欢迎推荐给您的朋友。
您的朋友报名成功后，您将获得¥100现金券，
可以直接抵扣下次学费。
被推荐的朋友也能享受¥100优惠，双赢！

这是推荐海报，您可以直接发朋友圈或发给朋友。"
```

### 3.3 体验课转化漏斗

```
流量层（100人）
├── 渠道: 地推、线上、转介绍
├── 动作: 9.9元体验课售卖
└── 目标: 收集线索

邀约层（60人）
├── 动作: 电话/微信确认时间
├── 提醒: 课前1天提醒
└── 目标: 确保到场

到场层（40人）
├── 动作: 2小时体验课交付
├── 关键: 让孩子兴奋、让家长看到价值
└── 目标: 建立信任

转化层（12人）
├── 动作: 现场优惠+限时逼单
├── 跟进: 未成交3天内回访
└── 目标: 30%转化率
```

### 3.4 社群运营策略

**家长群运营**:
```
建群时机: 体验课后，无论是否报名都邀请入群
群定位: 创客教育知识分享+学员作品展示+活动通知

内容节奏:
├── 周一: 本周课程预告
├── 周三: 科普知识分享（机器人、AI相关）
├── 周五: 学员作品展示（照片/视频）
├── 周末: 互动话题（如"你家孩子的创意时刻"）
└── 不定期: 福利活动（抽奖、免费体验名额）

转化动作:
├── 新班开班前: 限时早鸟价
├── 假期前: 集训营招募
└── 老学员续费前: 续费优惠+专属福利
```
