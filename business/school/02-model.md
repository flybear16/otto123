## 二、合作模式设计

### 2.1 三种合作模式对比

| 模式 | 学校投入 | 我方投入 | 收益分配 | 适合场景 | 推荐指数 |
|------|----------|----------|----------|----------|----------|
| **A. 纯服务外包** | 场地+组织 | 设备+师资+课程 | 学校付服务费¥8-12万/学期 | 学校预算充足，求省心 | ⭐⭐⭐⭐ |
| **B. 联合运营** | 场地+部分设备 | 课程+师资+运营 | 学费分成（学校30%，我方70%） | 学校愿意共担风险共享收益 | ⭐⭐⭐⭐⭐ |
| **C. 课程授权** | 场地+设备+本校老师 | 课程授权+师训 | 一次性授权费¥3-5万+年费¥1-2万 | 学校有师资，预算有限 | ⭐⭐⭐ |

### 2.2 主推模式：联合运营（B模式）

**核心逻辑**: 学校出场地和部分设备（已有电脑教室即可），我方输出完整课程和师资，学费按比例分成。

**为什么主推此模式**:
1. **学校接受度高**: 零现金投入，风险低
2. **我方现金流好**: 直接向家长收费，账期短
3. **利益绑定深**: 学校有动力配合招生和运营
4. **可复制性强**: 标准化后快速推广多校

**合作框架**:
```
┌─────────────────────────────────────────────────────┐
│                   联合运营合作协议                    │
├─────────────────────────────────────────────────────┤
│  甲方（学校）               乙方（我方）              │
│  ─────────────             ─────────────            │
│  • 提供社团课场地（计算机教室/创客空间）              │
│  • 提供现有电脑设备（学生自带或学校提供）             │
│  • 协助招生（发通知、家长会宣传）                    │
│  • 协助管理（考勤、安全、家长沟通）                  │
│                            • 提供OTTO机器人设备      │
│                            • 派遣专业师资授课        │
│                            • 提供完整课程体系        │
│                            • 负责课程运营和售后      │
├─────────────────────────────────────────────────────┤
│  收益分配: 学费总收入 × 学校30% + 我方70%            │
│  结算周期: 每月结算一次                              │
│  合作期限: 首签1学期，续签优先                       │
└─────────────────────────────────────────────────────┘
```

### 2.3 合作学校筛选标准

| 维度 | 硬性要求 | 加分项 |
|------|----------|--------|
| **学校类型** | 小学（3-6年级）或初中（7-8年级） | 科技特色学校、STEM示范校 |
| **学生规模** | 单年级≥200人 | 总人数≥1000人 |
| **场地条件** | 有计算机教室或可用教室≥1间 | 已有创客空间/机器人教室 |
| **家长付费能力** | 学区房价≥当地均价1.2倍 | 周边有高端社区 |
| **决策效率** | 能在2周内完成内部决策 | 校长直接分管科技教育 |
| **合作意愿** | 明确表达引入第三方意愿 | 已有其他素质类合作先例 |

**黑名单（不合作）**:
- 要求我方垫资采购学校设备
- 要求学费100%先进学校账户再分配
- 历史上有拖欠合作方款项记录
- 地理位置偏远（单程通勤>1小时）
