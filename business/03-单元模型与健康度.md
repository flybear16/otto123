# 单元模型与健康度

> 一堂五步法 · 第三步：商业模式
> 修炼水平：L4-L5（完整账 + 基准值）
> 核心方法：单元模型和 ROI · 科学决策三角形·深度

**L4 有完整账（宽度）→ L5 有基准值（深度）。** 本文将完整账本拆解为 8 大单元模型，并用行业基准值评估健康度。

## 维度一：按销售

### 1. 单订单模型

**定义**：每个学生的单次购买行为——一期课程、一套器材、一次集训。

#### 三种模式的订单结构

| 模式 | 订单金额 | 付款方 | 循环周期 | 决策链 |
|------|----------|--------|----------|--------|
| 学校社团课 | ¥1,280-1,580/学期 | 家长 | 2次/年（学期制） | 家长→学校→机构 |
| 晚托植入课 | ¥400/月（打包在晚托费中） | 家长 | 12次/年（月付） | 家长→晚托机构 |
| 周末班 | ¥80-150/课时，¥1,380/月 | 家长 | 12次/年（月付） | 家长直接决策 |

#### 订单循环分析

```
学校社团课订单循环（年周期）：
├── 春季学期 → 秋季学期 → 寒假集训 → 春季学期 ...
├── 循环周期：约4个月
├── 续费窗口：每学期末（约2周）
└── 转化漏斗：体验课(100%) → 报名(60%) → 续费(70%)

晚托植入课订单循环（月周期）：
├── 9月入学 → 连续月付 → 1月寒假暂停 → 2月恢复 → 7月暑假 ...
├── 循环周期：约1个月
├── 续费窗口：每月末（自动续费为主）
└── 转化漏斗：体验周(100%) → 报名(50%) → 连续3月(70%)

周末班订单循环（月/学期周期）：
├── 体验课 → 学期班 → 寒假集训 → 竞赛班 → 升级班 ...
├── 循环周期：约1-4个月（视产品线）
├── 续费窗口：每月末/学期末
└── 转化漏斗：体验课(100%) → 报名(30%) → 升级(40%)
```

**关键洞察**：
- 学校社团课的**决策链最长**（需学校配合），但**续费率最稳定**（学校背书）
- 晚托植入课的**循环最短**（月付），适合快速验证产品力
- 周末班的**客单价最高**、**LTV 最深**，但获客成本也最高

### 2. 单 SKU 模型

**定义**：每个产品线的独立盈利能力分析。

#### SKU 矩阵

| SKU | 定价 | 毛利率 | 角色 | 销量占比* | 利润贡献* |
|-----|------|--------|------|----------|-----------|
| 社团基础班 | ¥1,280/学期 | 35.7% | 引流+口碑 | 40% | 25% |
| 社团进阶班 | ¥1,580/学期 | 40%+ | 利润产品 | 20% | 18% |
| 晚托特色课 | ¥400/月 | 43.0% | 引流+差异化 | 15% | 12% |
| 周末学期班 | ¥1,380/月 | 16-28% | 利润产品 | 10% | 8% |
| 寒暑假集训 | ¥3,000-3,280/人 | 50%+ | 利润产品 | 8% | 20% |
| 竞赛集训班 | ¥2,280/人 | 55%+ | 口碑+品牌 | 3% | 8% |
| OTTO入门套件 | ¥330/台 | 24%（¥80利润） | 引流（硬件入口） | 3% | 3% |
| OTTO扩展模块 | ¥200-500/个 | 40%+ | 利润产品 | 1% | 6% |

*销量占比和利润贡献为估算，基于三种模式组合运营的假设。

#### SKU 角色分类

```
引流型（低毛利、高销量，负责获客）：
├── 社团基础班：进入学校场景的敲门砖
├── 晚托特色课：为晚托机构提供差异化卖点
└── OTTO入门套件：硬件入口，带动后续课程消费

利润型（高毛利，贡献主要利润）：
├── 寒暑假集训：集中时段、高客单、低边际成本
├── 竞赛集训班：品牌背书+高付费意愿
├── 社团进阶班：续费升级，毛利提升
└── 扩展模块：硬件复购，高毛利

防御型（低销量但战略价值）：
├── 体验课（¥29.9）：获客漏斗顶端
└── 校本课程定制：B端标杆效应
```

**关键洞察**：
- **社团课是流量入口**，毛利虽低（35.7%），但获客成本几乎为零
- **集训/竞赛是利润引擎**，寒暑假集训毛利 50%+，贡献约 28% 利润
- **硬件是沉默利润源**，扩展模块复购毛利 40%+，随课程深入自然发生

## 维度二：按人头

### 3. 单用户模型（C 端学生）

**定义**：单个学生从首次接触到流失的完整生命周期价值。

#### LTV 链路（社团课场景）

```
学生生命周期价值链（以社团课为起点）：

第1学期（首单）      第2学期（续费）    第2年（升级）     第3年（深度）
¥1,280              ¥1,580            ¥3,000+          ¥5,000+
├── 基础班30人       ├── 进阶班20人     ├── 寒假集训      ├── 竞赛集训
├── 购买套件¥330    ├── 购买扩展模块   ├── 购买新模块    ├── 参加多次竞赛
└── 转化率60%       └── 留存率70%      └── 升级率40%     └── 深度参与率25%

累计贡献估算：
├── 首单学生：¥1,280 + ¥330 = ¥1,610
├── 续费学生：¥1,610 + ¥1,580 + ¥200 = ¥3,390
├── 升级学生：¥3,390 + ¥3,000 + ¥500 = ¥6,890
└── 深度学生：¥6,890 + ¥5,000 + ¥800 = ¥12,690（估算）
```

#### LTV vs CAC 分析

| 指标 | 社团课 | 晚托植入 | 周末班 |
|------|--------|----------|--------|
| **CAC（获客成本）** | ¥50 | ¥30 | ¥500-800 |
| **首单金额** | ¥1,280 | ¥400/月 | ¥1,380/月 |
| **12个月留存率** | 70% | 60% | 50% |
| **3年LTV（估算）** | ¥6,890 | ¥5,760 | ¥8,280 |
| **LTV/CAC** | **137x** | **192x** | **10-17x** |
| **健康度** | 极优 | 极优 | 良好 |

> CAC 估算说明：社团课通过学校渠道获客，成本仅为 BD 拜访和材料费；晚托通过合作机构自然引流；周末班需独立获客。

#### 定量公式

```
LTV 计算公式：
LTV = ARPU × 毛利率 × 平均留存周期

示例（社团课）：
LTV = ¥1,580/学期 × 2学期 × 60%毛利率 × 3.5年 ≈ ¥6,636
（与分阶段估算 ¥6,890 基本吻合）

CAC 回收周期：
回收周期 = CAC / (ARPU × 毛利率)

示例（社团课）：
回收周期 = ¥50 / (¥948/年 × 60%) = 0.09年 ≈ 1个月
（即第一个月即可覆盖获客成本）

示例（周末班）：
回收周期 = ¥650 / (¥16,560/年 × 22%毛利率) = 0.18年 ≈ 2个月
```

### 4. 单客户模型（B 端学校/机构）

**定义**：每个 B 端合作方（学校、晚托机构）的生命周期和贡献值。

#### 学校客户模型

```
学校客户生命周期：

第1年（种子期）          第2年（验证期）         第3年（扩张期）
1个班/30人              2个班/50人              3个班/80人
├── 收入：¥40,000       ├── 收入：¥160,000     ├── 收入：¥350,000
├── 成本：¥95,000       ├── 成本：¥112,000     ├── 成本：¥140,000
├── 净贡献：-¥85,000    ├── 净贡献：¥48,000     ├── 净贡献：¥210,000
└── 状态：标杆投入      └── 状态：开始盈利      └── 状态：利润引擎

3年累计贡献：¥173,000/校
```

#### 晚托机构客户模型

```
晚托合作点生命周期：

第1个月（启动期）      第2-6个月（验证期）     第6个月+（稳定期）
1个班/20人            1-2个班/20-30人         持续运营
├── 月收入：¥8,000    ├── 月收入：¥12,000     ├── 月收入：¥12,000+
├── 月成本：¥6,000    ├── 月成本：¥5,000      ├── 月成本：¥4,500
├── 月毛利：¥2,000    ├── 月毛利：¥7,000      ├── 月毛利：¥7,500
└── 首期投入：¥3,300  └── 累计投入：¥3,300    └── 累计投入：¥3,300

回本周期：约1.4个月
合作点流失率估算：约20%/年（机构自身经营问题为主）
```

**关键洞察**：
- 学校客户**首年是亏损的**（标杆投入），但第2年起贡献加速
- 晚托合作点**回本极快**（1.4个月），但客户流失率较高（机构经营不稳定）
- **学校客户3年LTV ¥17.3万** 远超 **晚托合作点3年LTV ¥2.5万**，但获客难度也大得多

### 5. 单销售模型（BD/课程顾问）

**定义**：每个销售人员的产出效率。

#### BD 人效对比

| 指标 | 学校BD | 晚托BD | 周末班课程顾问 |
|------|--------|--------|----------------|
| 签约周期 | 2-3个月 | 2-4周 | 1-2周 |
| 年签约量 | 4-6校 | 15-20点 | 40-60人 |
| 单客户年贡献 | ¥10-20万 | ¥1.4万 | ¥1.7万 |
| 年创收 | ¥80-120万 | ¥30-40万 | ¥70-100万 |
| 年人力成本 | ¥12-15万 | ¥12-15万 | ¥10-12万 |
| 人效比 | 6-8x | 2-3x | 5-7x |
| 培养周期 | 3-6个月 | 1-2个月 | 1-2个月 |

```
人效比公式：
人效比 = 年创收 / 年人力成本

学校BD：¥100万 / ¥13.5万 = 7.4x ✅
晚托BD：¥35万 / ¥13.5万 = 2.6x ⚠️
周末班顾问：¥85万 / ¥11万 = 7.7x ✅

结论：
├── 学校BD人效最高，但培养周期长
├── 晚托BD人效最低，但签约最快、适合初期
└── 最佳策略：1人兼顾学校+晚托BD
```

### 6. 单履约模型（授课老师）

**定义**：每个授课老师的产出效率和服务质量。

#### 老师产能模型

| 指标 | 兼职老师 | 全职老师 |
|------|----------|----------|
| 月课时费 | ¥4,000-8,000 | ¥8,000-15,000 |
| 月创收能力 | ¥20,000-40,000 | ¥40,000-60,000 |
| 创收/成本比 | 3-5x | 3-4x |
| 管理学生数 | 60-90人 | 80-120人 |
| 培养周期 | 1-2个月（有基础） | 3-6个月 |
| 6个月留存率 | 50% | 80% |

```
师资产能公式：
创收/成本比 = (有效课时 × 课时费) / 月人力成本

兼职老师：(40课时 × ¥500) / ¥6,000 = 3.3x
全职老师：(80课时 × ¥500) / ¥12,000 = 3.3x

关键发现：
├── 创收/成本比相近，但全职老师稳定性远高于兼职
├── 兼职老师6个月留存率50%意味着年流失率极高
└── 扩张期必须转全职，否则履约质量无法保证

老师培养漏斗：
招聘(100人) → 面试(30人) → 培训(15人) → 试讲(10人) → 上岗(6人) → 优秀(2人)
                                                              ↓
                                                    培养成本 ¥2,000-5,000/人
```

## 维度三：按空间

### 7. 单门店/点位模型

**定义**：每个物理运营点位的经济模型。

#### 三种点位对比

| 指标 | 晚托植入点 | 周末校区 | 学校社团（单校） |
|------|-----------|----------|-----------------|
| **首期投入** | ¥3,300 | ¥200,000 | ¥95,000 |
| **月固定成本** | ¥1,560 | ¥63,000 | ¥0（学校提供场地） |
| **月变动成本** | ¥2,140 | ¥0 | ¥8,000（师资） |
| **月满员收入** | ¥8,200 | ¥75,000 | ¥23,000 |
| **月毛利** | ¥4,500 | ¥12,000 | ¥15,000 |
| **毛利率** | 55% | 16% | 65% |
| **回本周期** | 1.4个月 | 17个月 | 首年微利 |
| **运营复杂度** | 低 | 高 | 中 |

#### 点位扩张边际成本

```
晚托植入点（最轻资产）：
├── 第1个点：管理成本 ¥3,000/月
├── 第5个点：边际 +¥500/点/月
├── 第10个点：边际 +¥300/点/月（规模效应）
└── 第20个点：需专职运营，+¥5,000/月固定成本

学校社团（中资产）：
├── 单校首年投入 ¥95,000
├── 次年起运营成本 ¥9,000/月
├── 5校以内边际成本递减
└── 5校以上需增加研发投入

周末校区（最重资产）：
├── 第1个校区：固定成本 ¥63,000/月
├── 第2个校区：独立团队，¥60,000/月
└── 边际成本几乎不递减（每个校区独立运营）
```

### 8. 单城市模型

**定义**：单个城市的整体市场规模和运营目标。

#### 城市渗透路径

```
Year 1（种子期）—— 验证模型
├── 学校：3所（深度合作，建立标杆）
├── 晚托点：3-5个（植入模式）
├── 周末校区：0（暂不开设）
├── 覆盖学生：约200人
├── 年收入：约32万
└── 城市净贡献：-12万（投入期）

Year 2（验证期）—— 扩大规模
├── 学校：10所（标杆复制）
├── 晚托点：10个
├── 周末校区：1家（样板店）
├── 覆盖学生：约800人
├── 年收入：约133万
└── 城市净贡献：-3万（接近盈亏平衡）

Year 3（扩张期）—— 规模化
├── 学校：25所
├── 晚托点：20个
├── 周末校区：1-2家
├── 覆盖学生：约2,000人
├── 年收入：约365万
└── 城市净贡献：+56万（盈利）
```

#### 城市盈亏平衡

```
盈亏平衡公式：
盈亏平衡收入 = 年固定成本 / 毛利率

杭州城市：
= ¥840,000 / 0.6 = ¥140万/年

对应组合（Year 2末）：
├── 10所学校 × ¥10万/校 = ¥100万
├── 10个晚托点 × ¥3.2万/点 = ¥32万
├── 1家周末校区 × ¥90万/年 = ¥90万（估算）
└── 总计：约¥222万 > ¥140万 ✅ 盈利

结论：约13所学校 或 10校+10晚托点 即可覆盖城市运营成本
盈亏平衡周期：12-18个月
```

## 健康度总结

```
商业模式健康度：✅ 整体优秀（8/8 模型达标）

最强项（远超行业基准）：
├── LTV/CAC 比值 10-192x（行业 3-5x）→ 获客效率极高
├── 晚托植入回本 1.4个月（行业 12-24月）→ 最佳最小可复制单元
└── 学校毛利率 65%（行业 40-50%）→ 零场地成本优势

达标项（符合行业基准）：
├── 单订单循环频率 2-12次/年（≥2次/年）
├── 利润 SKU 占比 52%（≥50%）
├── B 端 3年 LTV ¥2.5-17.3万（≥5万）
├── BD 人效比 2-8x（≥3x）
├── 师资创收/成本比 3-5x（≥3x）
└── 城市盈亏平衡 12-18月（≤24月）

需关注：
├── 周末班回本 17个月（接近上限）
├── 兼职老师留存率 50%（扩张期风险）
└── 学校客户首年亏损（现金流压力）
```
