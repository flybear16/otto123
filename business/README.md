# 商业模式

OTTO 123 提供三种商业模式，适应不同资源条件和战略目标：

## 模式对比

| 模式 | 投入 | 风险 | 收益 | 适合人群 |
|------|------|------|------|----------|
| **学校社团课** | 低 | 低 | 中 | 有学校资源 |
| **晚托班合作** | 极低 | 极低 | 中 | 轻资产启动 |
| **C端周末班** | 高 | 高 | 高 | 有资金、想长期经营 |

## 详细方案

### 1. 学校社团课合作
- [查看完整方案](school.md)
- **核心逻辑**: 与中小学合作，输出课程和师资
- **定价**: ¥1,280-1,580/学期
- **收益**: 单校年利润 ¥50,000

### 2. 晚托班特色项目
- [查看完整方案](daycare.md)
- **核心逻辑**: 与晚托机构合作，植入OTTO课程
- **定价**: ¥300-500/月/人
- **收益**: 单点月净利润 ¥3,500

### 3. C端周末/假期班
- [查看完整方案](weekend.md)
- **核心逻辑**: 直接面向家长招生
- **定价**: ¥80-150/课时
- **收益**: 单校区年净利润 ¥144,000+

## 选择建议

```
资源条件                推荐模式
─────────────────────────────────────────
有学校资源              → 学校社团课
无场地、求快            → 晚托班合作
有资金、长期经营        → C端周末班
─────────────────────────────────────────
```

## 组合策略

也可以组合多种模式：
- **学校+晚托**: 学校建立品牌，晚托快速扩张
- **晚托+C端**: 晚托引流，C端深挖价值
- **三线并行**: 全面布局，互相导流
