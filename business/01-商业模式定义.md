# 商业模式定义

> 一堂五步法 · 第三步：商业模式
> 修炼水平：L1-L2（从不算账到算对账）
> 核心方法：单元模型和 ROI · L1-L2

---

## L1：算账意识——OTTO 靠什么赚钱？

**一句话定义**：OTTO 123 通过「开源硬件 + AI 语音 + 系统化课程」三件套，向中小学和晚托机构输出创客教育服务，从家长端收取课程费和器材费。

### 不是什么

- 不是卖硬件的公司（硬件是入口，课程是利润）
- 不是纯 C 端培训机构（B 端渠道获客成本更低）
- 不是 SaaS 平台（教育需要线下交付和师资）

### 是什么

- **本质**：教育服务公司，用硬件和软件降低师资门槛
- **收入来源**：课程服务费（50%）+ 器材销售费（30%）+ 定制开发费（15%）+ 竞赛服务费（5%）
- **付费方**：家长（最终付费）→ 学校/晚托机构（渠道）→ OTTO（服务提供方）

## L2：选对单元模型——三种商业模式全景

### 为什么需要三种模式？

教育行业的单元模型不是单一的——不同渠道的获客成本、履约成本、LTV 完全不同。只看一种模式会「算错账」。

### 模式一：学校社团课（B2B2C）

```
商业模式：
学校（出场地+渠道） ← OTTO（出课程+师资+器材） ← 家长（付费）

合作模式：联合运营
├── 学校提供：场地、学生、课时（课后服务时间）
├── OTTO 提供：课程体系、授课教师、教学设备
└── 家长付费：¥1,280-1,580/学期（16 课时）
```

| 维度 | 数值 |
|------|------|
| 目标客户 | 小学 3-6 年级、初中 7-8 年级 |
| 付费方 | 家长（通过学校代收或直接缴费） |
| 单班人数 | 30 人 |
| 单班收入 | ¥1,280/人 × 30 = ¥38,400/学期 |
| 场地成本 | ¥0（学校提供） |
| 首年投入 | ¥95,000/校（器材+课程+师训） |
| 3 年累计毛利 | ¥384,000/校 |
| 核心优势 | 零场地成本、品牌背书、批量获客 |

### 模式二：晚托班合作（B2B2C）

```
商业模式：
晚托机构（出场地+学生） ← OTTO（出特色课程+教师） ← 家长（打包付费）

合作模式：课程植入
├── 晚托机构提供：场地、现有学生、日常运营
├── OTTO 提供：OTTO 特色课程、师资（可选）
└── 家长付费：¥300-500/月（打包在晚托费中或单独收取）
```

| 维度 | 数值 |
|------|------|
| 目标客户 | 双职工家庭小学生 |
| 付费方 | 家长（通过晚托机构代收） |
| 单班人数 | 20 人 |
| 单点月收入 | ¥400/人 × 20 = ¥8,000 |
| 场地成本 | ¥0（晚托机构提供） |
| 首期投入 | ¥3,300/点（基础设备） |
| 回本周期 | 1.4 个月 |
| 核心优势 | 极轻资产、最快验证、快速铺量 |

### 模式三：C 端周末/假期班（B2C）

```
商业模式：
OTTO（全链条运营） ← 家长（直接付费）

运营模式：直营
├── OTTO 提供：场地、课程、师资、设备、运营
├── 家长直接决策：体验课 → 正式报名
└── 定价：¥80-150/课时，¥1,380/月
```

| 维度 | 数值 |
|------|------|
| 目标客户 | 对科技教育感兴趣的家庭 |
| 付费方 | 家长（直接付费） |
| 单班人数 | 10-15 人（小班制） |
| 单校区月收入 | ¥75,000（满员 60 人） |
| 场地成本 | ¥30,000/月（租赁） |
| 首期投入 | ¥200,000（装修+设备+押金） |
| 回本周期 | 17 个月 |
| 核心优势 | 利润高、品牌直营、用户粘性强 |

## 三种模式对比

| 维度 | 学校社团课 | 晚托植入 | C 端周末班 |
|------|-----------|---------|-----------|
| **模式** | B2B2C | B2B2C | B2C |
| **场地** | 学校提供 | 晚托提供 | 自租 |
| **投入** | ¥95,000/校 | ¥3,300/点 | ¥200,000/校 |
| **回本** | 首年微利 | 1.4 个月 | 17 个月 |
| **毛利率** | 65% | 55% | 16-28% |
| **LTV/CAC** | 137x | 192x | 10-17x |
| **月覆盖学生** | 30 人 | 20 人 | 60 人 |
| **决策链** | 长（需学校配合） | 短（机构决策） | 中（家长决策） |
| **扩张速度** | 慢（2-3 月/校） | 快（2-4 周/点） | 慢（需建校区） |
| **适合阶段** | 验证期、扩张期 | 启动期 | 扩张期 |

## 收入结构分析

### 四大收入来源

```
收入构成（3 年目标）：
├── 课程服务收入：50%
│   ├── 学期班（学校社团、周末班）
│   ├── 寒暑假集训营
│   └── 竞赛集训班
├── 器材销售收入：30%
│   ├── OTTO 入门套件（¥330/台）
│   ├── 扩展模块（¥200-500/个）
│   └── DIY 配件
├── 定制开发收入：15%
│   ├── 校本课程定制
│   └── 特色方案开发
└── 竞赛/其他收入：5%
    ├── 竞赛报名/指导费
    └── 考级/认证费
```

### 收入特性

| 收入类型 | 毛利率 | 可持续性 | 现金流 | 战略角色 |
|----------|--------|----------|--------|----------|
| 课程服务 | 60% | 高（学期制循环） | 好（预收款） | 利润引擎 |
| 器材销售 | 24-40% | 中（一次性） | 一般 | 入口+利润补充 |
| 定制开发 | 50%+ | 中（项目制） | 较好 | 标杆+品牌 |
| 竞赛服务 | 55%+ | 中（季节性） | 好 | 口碑+引流 |

## L2 常见错误避坑

### 坑 1：选错单元模型

> 教育行业不是单次交易，是 LTV 游戏。

错误做法：只看单节课的利润（如 ¥100 课时费 - ¥50 成本 = ¥50 利润），忽略续费率和 LTV。

正确做法：以「单用户 LTV」为单元模型。一个社团课学生 3 年 LTV ¥6,890，远比课时费重要。

### 坑 2：只算大账

> 不能只看年收入 416 万，要看单点经济模型。

错误做法：只算「25 所学校 × 11.6 万/校 = 290 万」，忽略单校首年亏损 ¥85,000。

正确做法：先算清最小可复制单元（晚托植入点：投入 ¥3,300、回本 1.4 个月），再乘以扩张系数。

### 坑 3：存在盲区

> 忽略隐形成本是算错账的主因。

常见盲区：
- 师资培养成本（单人 ¥2,000-5,000，6 个月留存率 50%）
- 设备折旧和损耗（年损耗率约 15%）
- 管理成本随规模非线性增长（10 个点以上需加管理层级）
- 资金占用成本（学校客户首年亏损需要缓冲资金）

## 模式组合策略

### 按阶段选择

```
启动期（Year 1）—— 晚托为主
├── 目标：验证产品力、积累教学经验
├── 策略：3-5 个晚托植入点，快速试错
├── 投入：¥15,000-20,000
└── 预期：月净利润 ¥10,000-15,000

验证期（Year 2）—— 学校为主
├── 目标：建立品牌、验证 B 端模型
├── 策略：10 所学校 + 10 个晚托点
├── 投入：¥100,000+（学校标杆投入）
└── 预期：接近盈亏平衡

扩张期（Year 3）—— 三线并行
├── 目标：规模化盈利
├── 策略：25 所学校 + 20 个晚托点 + 1-2 家周末校区
├── 投入：自给自足
└── 预期：年净利润 ¥560,000
```

### 组合逻辑

```
晚托 → 学校（晚托验证产品力 → 用案例打动学校）
学校 → 周末（学校建立品牌 → 家长主动找周末班）
周末 → 竞赛（周末班发现好苗子 → 竞赛出成绩 → 反哺品牌）
```
