# 核心价值主张

> OTTO 123 用 1/3 的价格、开源的方式，让每个孩子都能拥有一个会说话、会跳舞、能编程、能改装的 AI 机器人。

---

## 差异化定位

### 四大核心差异化要素

```
OTTO 123 的独特定位：

低成本          ──  单台 ≤ 250 元，仅为商业产品的 1/5 ~ 1/10
AI 语音交互     ──  接入大模型，自然语言对话 + 语音控制动作
开源硬件        ──  基于 Otto DIY 二次开发，硬件设计完全开放
DIY 扩展       ──  3D 打印外壳 + 标准化扩展接口，无限可能
```

> 我们不是在做一个更便宜的乐高，而是在做一个**更开放、更智能、更触手可及**的创客教育平台。

### 定位一句话

**"让每个孩子都能玩得起、造得出、赛得赢的 AI 机器人。"**

## 对不同用户的价值

### 对学生：从消费者到创造者

| 价值维度 | OTTO 123 的价值 |
|----------|-----------------|
| **趣味体验** | AI 语音对话 + 动作编程 + DIY 外壳，三重乐趣叠加 |
| **低门槛创造** | 图形化编程拖拽即用，不需要先学代码 |
| **个性化表达** | 3D 打印自定义外壳，每个孩子的机器人都是独一无二的 |
| **竞赛激励** | 机构内部赛 → 机构间联赛 → 线上社区，持续的学习动力 |
| **团队协作** | 2-3 人共用一台机器人，培养合作能力 |

### 对教师：开箱即用的教学系统

| 价值维度 | OTTO 123 的价值 |

## 价值主张画布

### 用户侧（Customer Profile）

| 维度 | 内容 |
|------|------|
| **用户工作** | 晚托班授课、课后社团活动、校内兴趣课 |
| **用户痛点** | ① 硬件成本高，机构难以负担 ② 缺乏差异化课程 ③ 内容安全无法保障 ④ 教师备课负担重 ⑤ 学生缺乏动手实践机会 |
| **用户收益** | ① 低成本引入特色课程 ② 学生有成就感和持续动力 ③ 家长透明了解学习情况 ④ 教师有现成教案可用 ⑤ 竞赛体系提供展示出口 |

### 产品侧（Value Map）

| 维度 | 内容 |
|------|------|
| **产品与服务** | OTTO 开源机器人硬件 + 图形化编程平台 + AI 语音交互系统 + 竞赛管理平台 + 多角色后台管理系统 |
| **痛点缓解** | ① 单台 ≤ 250 元，大幅降低硬件门槛 ② 内置 12 节课程，零备课 ③ 多层安全审计保障内容安全 ④ OTA 升级 + 故障诊断降低运维负担 ⑤ 课堂模式工具保障教学秩序 |
| **收益创造** | ① 学生从"消费者"变为"创造者" ② 机构获得差异化竞争力 ③ 教师教学效率提升 3-5 倍 ④ 家长获得透明反馈 ⑤ 竞赛获奖提升品牌价值 |

### 价值匹配关系

```
用户痛点                  →  产品解决方案
─────────────────────────────────────────
硬件成本高                 →  开源设计，单台 ≤ 250 元
缺乏差异化课程             →  AI 机器人 + 竞赛体系，独特卖点
内容安全无法保障           →  敏感词 + AI 语义多层审计
教师备课负担重             →  12 节内置教程 + 推送资源库
学生缺乏动手实践           →  DIY 外壳 + 动作编程 + 硬件扩展
家长不了解学习情况          →  对话摘要 + 使用数据，手机端可见
```
