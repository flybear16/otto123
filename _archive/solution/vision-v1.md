# 愿景与使命

## 愿景（Vision）

> **让每个孩子都能创造自己的机器人**

我们相信，创造能力不是少数人的天赋，而是每个孩子都拥有的潜能。

在 AI 时代，编程和机器人素养将成为像阅读、写作一样的基础能力。
OTTO 123 致力于降低创客教育的门槛，让优质科技教育触手可及。

## 使命（Mission）

### 短期（1-2年）

- 建立标准化的 OTTO 机器人课程体系
- 验证三种商业模式的可行性（学校/晚托/C端）
- 服务 1000+ 学生，积累教学数据和案例

### 中期（3-5年）

- 成为区域领先的创客教育品牌
- 建立完善的师资培训和认证体系
- 输出课程授权，合作 100+ 学校/机构
- 培养学生在市级、省级科技竞赛中获奖

### 长期（5-10年）

- 推动开源硬件在教育领域的普及
- 建立全国性的创客教育社区
- 影响 100万+ 孩子的科技启蒙
- 培养下一代科技创新人才

## 教育理念

### 从"教知识"到"育能力"

| 传统教育 | OTTO 123 教育 |
|----------|---------------|
| 老师讲授，学生听讲 | 项目驱动，探索学习 |
| 标准答案，统一评价 | 多元解法，过程评价 |
| 记忆知识，应付考试 | 解决问题，培养思维 |
| 被动接受，害怕犯错 | 主动尝试，从错误学习 |

### STEM 教育框架

```
S - Science（科学）
    └── 机器人运动原理、电路知识、传感器原理

T - Technology（技术）
    └── 编程技术、3D打印、电子制作

E - Engineering（工程）
    └── 机械设计、系统集成、调试优化

M - Mathematics（数学）
    └── 角度计算、坐标系统、逻辑推理

+ A - Arts（艺术）
    └── 外观设计、舞蹈编排、创意表达

= STEAM 综合素养
```

## 社会价值

### 教育公平

- 降低创客教育门槛，让二三线城市孩子也能接触优质科技教育
- 开源硬件成本仅为商业产品的 1/5-1/10
- 课程资源免费开放，降低师资依赖

### 人才培养

- 培养下一代科技创新人才
- 提升中国青少年的工程素养和创造力
- 为科技特长生选拔输送人才

### 产业推动

- 推动开源硬件在教育领域的应用
- 培育创客教育市场
- 促进教育科技产业发展
