# Ideation: Otto Robot 教育平台

## 背景

基于 Otto DIY 开源机器人（ESP32 + 小智AI）进行二次研发，打造面向初中晚托班的 AI 人形机器人教育平台。全栈自研（硬件改版 + 固件 + 云端 + 前端）。

## 原始点子（来自飞熊）

### 1. 语音交互
- 连接大模型后，通过语音对话实现跳舞、回答问题等功能

### 2. 网页端配置
- 提供可视化界面，方便用户拼装和调试

### 3. 硬件 DIY 扩展
- **外壳自定义**：突破固定形态，支持芭比娃娃造型、高达机器人造型等个性化外壳
- **摄像头模块**：增加视觉识别能力
- **大尺寸显示屏**：支持字幕显示，丰富交互体验

### 4. 动作编程与训练
- 通过网页端调整机器人步伐
- 训练多样化的舞步和动作
- 作为课程内容的一部分，持续升级

### 5. 创意比拼与竞赛机制
- **外观选美**：同学之间展示 DIY 外壳，比拼谁的更漂亮
- **动作技能展示**：展现不同的动作编程能力
- **功能创新竞赛**：互相交流功能扩展思路（加装显示器、增加舵机等）

### 6. 权限分级管理
- 在现有后台页面基础上，增加家长端权限
- 家长可定向植入特定内容
- 实现内容精准推送与管控
- 为孩子提供更安全的使用体验

## 头脑风暴补充（通过问答确认）

| 问题 | 决策 |
|------|------|
| 目标用户 | 教培机构，从初中晚托班开始 |
| 机器人角色 | 兴趣课（核心）+ 课间娱乐 + 机构引流 |
| 硬件模式 | 机构提供公用 + 学生可自购带回家 |
| 研发范围 | 全栈自研（硬件/固件/云端/前端） |
| 竞赛机制 | 三期：机构内部 → 机构间联赛 → 线上社区 |
| 家长管控 | 教学内容推送 + 家长推送 + 安全过滤 |
| 商业模式 | 待定，PRD 不限定 |
