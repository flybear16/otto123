## 一、财务模型假设

### 1.1 核心假设

| 假设项            | 数值            | 说明            |
| -------------- | ------------- | ------------- |
| **客单价（学期课程）**  | 4000 元/班       | 16 次课，20-30 人/班 |
| **器材单价（入门套件）** | 1000 元/套       | 含机器人 + 基础传感器    |
| **毛利率（课程）**    | 60%           | 扣除师资、场地、运营    |
| **毛利率（器材）**    | 40%           | 含定制开发摊销       |
| **学校续约率**      | 70%           | 行业保守估计        |
| **单校扩张速度**     | 首年 1 个班 → 次年 2 个班 | 口碑驱动          |
| **人效（课程顾问）**   | 管理 5 所学校/人      | 含 BD+ 服务        |

### 1.2 收入结构

```
收入构成（3年目标）：
├── 课程服务收入：50%
├── 器材销售收入：30%
├── 定制开发收入：15%
└── 竞赛/其他收入：5%

收入特性：
├── 课程：高毛利、可持续、现金流好
├── 器材：一次性、带动后续服务
├── 定制：高客单、标杆效应
└── 竞赛：口碑、引流、品牌
```

## 四、敏感性分析

### 4.1 关键变量影响

| 变量       | 基准值     | 变化            | 对 Year 3 净利润影响 |
| -------- | ------- | ------------- | ------------ |
| **学校数量** | 25 所     | +5 所（+20%）     | +11.2 万（+20%） |
|          |         | -5 所（-20%）     | -11.2 万（-20%） |
| **客单价**  | 4000 元/班 | +500 元（+12.5%） | +11.2 万（+20%） |
|          |         | -500 元（-12.5%） | -11.2 万（-20%） |
| **毛利率**  | 60%     | +5%           | +14.5 万（+26%） |
|          |         | -5%           | -14.5 万（-26%） |
| **续约率**  | 70%     | +10%          | +5.6 万（+10%）  |
|          |         | -10%          | -8.4 万（-15%）  |

### 4.2 盈亏平衡点

```
盈亏平衡分析（Year 3）：

固定成本：约90万/年（人员+运营）
变动成本率：约40%
毛利率：60%

盈亏平衡收入 = 固定成本 / 毛利率
            = 90万 / 0.6
            = 150万/年

对应学校数：约13所学校（每校平均11.5万收入）

结论：
├── Year 1：亏损正常（投入期）
├── Year 2：接近盈亏平衡
└── Year 3：大幅盈利（25所学校）
```

| ----------- | ------ | ----------- |

## 七、财务模型总结

```
财务可行性：✅ 可行

关键数字：
├── 启动资金：25-35万
```

### 7.2 风险提示

| 风险       | 影响     | 应对          |
| -------- | ------ | ----------- |
| 学校拓展慢于预期 | 亏损期延长  | 控制成本，单点深耕   |
| 价格竞争激烈   | 毛利率下降  | 强化差异化，不打价格战 |
| 师资供给不足   | 服务质量下降 | 建立师训体系，提高人效 |
| 政策变化     | 市场需求萎缩 | 关注政策，多元布局   |
