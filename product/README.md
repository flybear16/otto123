# 产品方案

## 产品体系

OTTO 123 产品体系包含三个层次：

```
硬件层: OTTO Robot 开源套件
        └── ESP32S3主控、舵机、麦克风、扬声器
        
课程层: 系统化创客课程
        └── 基础班、进阶班、竞赛班、假期营
        
服务层: 完整学习支持
        └── 教学服务、家长沟通、成长跟踪
```

## 硬件产品

### OTTO基础套件
- ESP32S3主控板（预刷固件）
- 4个舵机（已校准）
- 麦克风+扬声器模块
- 3D打印外壳（基础版+空白DIY版）
- 锂电池+充电器
- 组装说明书（图文+视频）

### OTTO扩展包
- 摄像头模块（OV2640）
- 1.54寸显示屏模块
- 扩展支架和连接线
- 人脸识别示例代码
- 表情动画素材包
