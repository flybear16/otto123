# 增长

> 一堂五步法 · 第四步：增长
> 核心方法：增长高手修炼地图 L1-L6 + 增长周期模型

**一句话定义**：OTTO 123 通过「学校社团课 → 晚托植入 → 周末校区」三渠道递进，从获客增长到系统增长再到对抗增长，建立可持续的教育服务增长模型。

## 增长周期模型

> 参考原文：[一堂五步法·增长周期模型](/yitang/一堂五步法-增长周期模型.md)

### 三个关键窗口

| 窗口 | OTTO 现状 | 应对策略 |
|------|----------|----------|
| **流量窗口** | 双减政策红利（2022-2028）+ 新课标落地（2023-2030） | 抢占学校渠道，建立素质教育品牌 |
| **竞争窗口** | 初中机器人教育市场几乎空白 | 乐高/Makeblock 主攻小学，OTTO 差异化定位初中 |
| **资本窗口** | 启动资金 25-35 万，Year 3 自给自足 | 先验证模型再融资，保持控制权 |

### 周期 1：获客增长（复制单元模型）

> 核心：找到一个可持续的渠道

**OTTO 对应**：Year 1，通过晚托植入和学校社团课验证获客渠道。

```
渠道预判 → 渠道探索 → 销售预判 → 销售搭建 → 选品选址
    │           │           │           │           │
    ▼           ▼           ▼           ▼           ▼
 晚托/学校    3校试点     销转率≥30%   BD体系搭建   课时/器材定价
```

**典型卡点与应对**：

| 卡点 | OTTO 风险 | 应对 |
|------|----------|------|
| 非标渠道 | 依赖个人关系拿学校资源 | 建立 BD SOP，可复制化 |
| 渠道错配 | 初中生家长付费意愿 vs 小学生不同 | 竞赛出口+中考科技特长生政策验证 |
| 销售太弱 | 体验课转化率低 | 标准化体验课流程 + 教师培训 |

**最糟结果**：无法复制 → 应对：控制首年投入 ≤20 万

→ 详见 [开业筹备](/growth/startup.md)

### 周期 2：系统增长（细化单元模型）

> 核心：找到一个稳定增长模型

**OTTO 对应**：Year 2，建立增长指标体系和运营系统。

```
增长指标 → 业务公式 → 找驱动力 → 假设验证 → 转化提升
    │          │          │          │          │
    ▼          ▼          ▼          ▼          ▼
 十大指标   收入=校×班×人  续费率/捕获率  A/B测试    销转率60%+
```

**典型卡点与应对**：

| 卡点 | OTTO 风险 | 应对 |
|------|----------|------|
| 指标跑偏 | 关注注册数而非续费率 | 严格按十大典型指标监控 |
| 缺少模型 | 没有量化的增长公式 | 单校经济模型 + 敏感性分析 |
| 缺少假设 | 凭直觉调整课程/定价 | 每次调整设定假设+验证周期 |

**最糟结果**：现金流崩 → 应对：保持 6 个月运营资金储备

→ 详见 [日常运营](/growth/daily.md) · [师资管理](/growth/teachers.md) · [家长服务](/growth/parents.md)

### 周期 3：对抗增长（对抗单元模型）

> 核心：持续对抗持续拉升增长

**OTTO 对应**：Year 3+，面对竞争加剧和规模扩张的挑战。

```
构建优势 → 对抗劣势 → 工业生产 → 增长飞轮 → 练基本功
    │          │          │          │          │
    ▼          ▼          ▼          ▼          ▼
 课程IP    价格/品牌    师训体系   口碑→获客    教学质量
```

**典型卡点与应对**：

| 卡点 | OTTO 风险 | 应对 |
|------|----------|------|
| 忽略壁垒 | 课程被抄袭、硬件被模仿 | 开源硬件+课程IP+师训体系三重壁垒 |
| 不擅对抗 | 乐高/Makeblock 进入初中市场 | 加速占领学校渠道，建立先发优势 |
| 过早分心 | Year 1 就想做周末校区 | 严格按里程碑推进，学校模式先验证 |

**最糟结果**：越大越亏 → 应对：规模扩张前确保单点盈利

→ 详见 [课程体系框架](/growth/curriculum-framework.md) · [安全管理](/growth/safety.md)

## OTTO 增长节奏

| 阶段 | 时间 | 增长周期 | 核心动作 | 关键指标 |
|------|------|----------|----------|----------|
| 种子期 | Year 1 | 周期1·获客增长 | 晚托验证 → 3 校签约 | 捕获率≥10%、销转率≥30% |
| 验证期 | Year 2 | 周期2·系统增长 | 10 校扩张 → 指标体系 | 续费率≥60%、完课率≥80% |
| 扩张期 | Year 3 | 周期3·对抗增长 | 25 校 → 周末校区试点 | 净利润 56 万、推荐率≥25% |
| 深耕期 | Year 4 | 周期3·对抗增长 | 50 校 → 多城复制 | 净利率 20-25% |
