## 运营原则

```
安全第一  ·  质量为本  ·  服务至上  ·  持续改进
```

## 关键运营指标

| 指标 | 目标值 | 监控频率 |
|------|--------|----------|
| 出勤率 | >90% | 每日 |
| 满意度 | >4.5/5 | 每月 |
| 续费率 | >70% | 每学期 |
| 转介绍率 | >30% | 每学期 |
| 体验课转化率 | >25% | 每月 |
| 投诉率 | <2% | 每月 |
