## 产品概述

OTTO 123 产品体系包含三个层次：

```
硬件层: OTTO Robot 开源套件
        └── ESP32S3主控、舵机、麦克风、扬声器
        
课程层: 系统化创客课程
        └── 基础班、进阶班、竞赛班、假期营
        
服务层: 完整学习支持
        └── 教学服务、家长沟通、成长跟踪
```

## 核心产品包

| 产品 | 包含内容 | 适用场景 |
|------|----------|----------|
| **OTTO基础套件** | 主控板+舵机+外壳+电池 | 入门首选 |
| **OTTO扩展包** | 摄像头+显示屏+传感器 | 进阶学习 |
| **课程授权包** | 教案+PPT+视频+师训 | 机构合作 |
| **竞赛辅导包** | 专项训练+作品指导 | 竞赛冲刺 |
