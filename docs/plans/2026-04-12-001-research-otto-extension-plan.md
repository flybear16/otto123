# 调研 OTTO 相关扩展方案

## Overview

本调研旨在梳理 OTTO Robot 相关的扩展方案，包括 3D 打印配件、舵机升级、屏幕扩展、摄像头模块、以及其他扩展接口。为 OTTO 123 项目在硬件扩展方向提供技术参考和选型依据。

## 研究范围

### 1. 3D 打印扩展

* OTTO 机器人现有 3D 打印部件清单
* 社区衍生设计（Thingiverse、Printables 等）
* 功能性扩展配件（夹爪、传感器支架等）
* 外观定制化方案

### 2. 舵机升级方案

* 标准舵机 vs 金属齿轮舵机对比
* 力矩要求分析
* 候选型号推荐（MG90S, MG996R, LD-20HB 等）
* 舵机控制接口与供电要求

### 3. 屏幕扩展

* I2C OLED/LCD 屏幕接口
* 推荐屏幕型号（0.96" OLED, 1.3" SH1106, 1.44" TFT 等）
* 屏幕固定方案
* 图形库支持情况

### 4. 摄像头模块

* 适合 OTTO 的小型摄像头
* ESP32-CAM 集成方案
* 图像识别功能扩展
* 视频流输出能力

### 5. 其他扩展接口

* I2C 传感器扩展
* 舵机扩展板方案
* 电源管理方案
* 通信接口（蓝牙、WiFi）

## 预期产出

创建文档：`otto-extension-research.md`，包含：

* 各扩展类型的技术对比表
* 推荐方案及理由
* 接线图/接口说明
* 采购参考链接

## 研究方式

1. 搜索 OTTO Robot 官方文档和社区
2. 搜索 GitHub 上的 OTTO 相关项目
3. 搜索国内论坛（CSDN、知乎、立创开源社区）
4. 整理扩展方案对比

## 分支研究任务

由于各扩展类型相互独立，建议并行研究：

* 并行任务 1:3D 打印 + 外观扩展
* 并行任务 2：舵机 + 电源管理
* 并行任务 3：屏幕 + I2C 扩展
* 并行任务 4：摄像头 + 通信模块
