# 一堂五步法·增长周期模型

> 来源：一堂课程体系 · 五步法·第四步：增长
> 原始图片：https://cdn.yitang.top/localfile/prod/00802d57a885185632b19583e7d955a4.png

增长周期模型以横向时间轴分为三个阶段，每个阶段对应不同的增长驱动逻辑和重点工作。

---

## 三个关键窗口

| 窗口 | 说明 |
|------|------|
| **流量窗口** | 特定渠道的流量红利期，窗口关闭后获客成本急剧上升 |
| **竞争窗口** | 市场竞争格局的空白期，窗口关闭后被竞争对手挤压 |
| **资本窗口** | 融资环境的友好期，窗口关闭后融资难度和成本增加 |

---

## 周期 1：获客增长（复制单元模型）

> 核心：找到一个可持续的渠道

**驱动逻辑**：获客驱动

**重点工作**：

```
渠道预判 → 渠道探索 → 销售预判 → 销售搭建 → 选品选址
```

**示意图**：线性结构，代表从 0 到 1 的复制过程

**典型卡点**：

| 卡点 | 表现 | 后果 |
|------|------|------|
| 非标渠道 | 依赖不可复制的获客方式（如个人关系、一次性机会） | 无法复制 |
| 渠道错配 | 选择的渠道与目标用户不匹配 | 获客效率低 |
| 销售太弱 | 缺乏销售能力或销售体系搭建不完善 | 转化率低 |

**最糟结果**：无法复制

---

## 周期 2：系统增长（细化单元模型）

> 核心：找到一个稳定增长模型

**驱动逻辑**：系统驱动

**重点工作**：

```
增长指标 → 业务公式 → 找驱动力 → 假设验证 → 转化提升
```

**示意图**：树状结构，代表系统化的增长模型

**典型卡点**：

| 卡点 | 表现 | 后果 |
|------|------|------|
| 指标跑偏 | 关注虚荣指标（注册数、曝光量）而非核心指标 | 方向错误 |
| 缺少模型 | 没有建立可量化的增长公式和模型 | 无法优化 |
| 缺少假设 | 凭直觉决策，缺乏假设验证的实验思维 | 效率低下 |

**最糟结果**：现金流崩

---

## 周期 3：对抗增长（对抗单元模型）

> 核心：持续对抗持续拉升增长

**驱动逻辑**：对抗驱动

**重点工作**：

```
构建优势 → 对抗劣势 → 工业生产 → 增长飞轮 → 练基本功
```

**示意图**：复杂树状结构，代表多维度的对抗体系

**典型卡点**：

| 卡点 | 表现 | 后果 |
|------|------|------|
| 忽略壁垒 | 不注重构建竞争壁垒，只关注规模扩张 | 易被复制 |
| 不擅对抗 | 缺乏与竞争对手正面竞争的策略和手段 | 市场份额被蚕食 |
| 过早分心 | 核心业务未稳固就分散精力到新业务 | 越大越亏 |

**最糟结果**：越大越亏

---

## 纵向维度

每个周期都需要从以下维度进行思考：

| 维度 | 说明 |
|------|------|
| **盯紧市场** | 关注市场变化、用户需求变化、竞争格局变化 |
| **抓主要矛盾** | 识别当前阶段最核心的增长瓶颈，集中力量突破 |
| **重点工作** | 每个阶段有明确的重点工作清单，按优先级执行 |
