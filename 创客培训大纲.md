# Otto Robot 创客培训大纲

> 适用对象：初中生
> 设计日期：2026-05-15

---

## 第一步：开箱认亲 🤝（20分钟）

- 认识 Otto 的各个部件（舵机、主板、传感器、超声波模块）
- 了解它能做什么：走路、跳舞、避障、唱歌
- 看一个成品 Otto 演示视频，激发兴趣

## 第二步：组装大作战 🔧（40-60分钟）

- 按照图纸拼装 Otto（这步很有成就感）
- 重点理解：结构搭建 → 电子连接 → 通电测试
- 教学点：耐心看图、工具使用、硬件连接基础

## 第三步：编程初体验 🎮（40分钟）

- 用 mBlock / Scratch 图形化编程控制 Otto
- 让它走两步、转个弯、打个招呼
- 教学点：程序逻辑、序列、循环

## 第四步：原理深挖 🧠（30分钟）

- 为什么舵机角度决定走路姿势？
- 超声波传感器是怎么"看见"障碍物的？
- 引导探究：硬件 → 软件的映射关系

## 第五步：创意挑战赛 🏆（60-90分钟）

- 任务卡形式：比如"让 Otto 走一个正方形"、"做一个会跳舞的 Otto"
- 分组竞赛/合作，融合创意和工程思维
- 培养问题拆解、迭代调试能力

## 第六步：展示与复盘 📢（20分钟）

- 每人/每组演示自己的 Otto 作品
- 分享：遇到了什么问题？怎么解决的？
- 梳理知识点，引出进阶方向

---

## 教学设计要点

| 要素 | 建议 |
|------|------|
| **时长** | 2-4小时为宜，分2-3次完成 |
| **分组** | 2-3人一组，培养协作 |
| **挑战梯度** | 先模仿，再改写，最后原创 |
| **成就感** | 每个阶段留一个"小作品"，强化正反馈 |
| **引导方式** | 启发式提问 > 直接给答案 |
