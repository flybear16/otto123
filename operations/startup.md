# 开业筹备 - 从0到1开店指南

> 版本: v1.0 | 创建日期: 2026-04-04 | 作者: OTTO 运营团队

## 一、前期筹备时间线

```
Week 1-2: 项目启动
├── 市场调研（周边学校、竞品分析、目标用户画像）
│   └── 输出: 市场调研报告、选址评估表
├── 资金规划（启动资金、运营储备金、设备采购预算）
│   └── 输出: 资金预算表、现金流预测（6个月）
└── 团队组建（确定店长、核心教师）
    └── 输出: 组织架构图、人员到岗计划

Week 3-4: 场地落地
├── 选址签约（实地考察、租金谈判、合同签订）
│   └── 输出: 租赁合同、场地平面图
├── 装修施工（功能分区、弱电布线、安全设施）
│   └── 输出: 装修方案、施工时间表
└── 资质办理（营业执照、消防审批、办学许可）
    └── 输出: 资质办理进度表

Week 5-6: 设备与环境
├── 设备采购（OTTO机器人套件、教学平板、网络设备）
│   └── 输出: 设备采购清单、到货验收单
├── 网络部署（5GHz Wi-Fi AP、带宽测试、信号覆盖验证）
│   └── 输出: 网络验收报告
└── 环境布置（教学区、体验区、家长休息区、设备存放区）
    └── 输出: 场地布置方案

Week 7: 人员准备
├── 教师招聘与培训（面试、试讲、设备操作培训）
│   └── 输出: 教师档案、培训考核记录
├── 运营人员到岗（前台、教务、市场推广）
│   └── 输出: 岗前培训记录
└── 管理系统搭建（排课系统、学员管理、设备管理后台）
    └── 输出: 系统配置完成确认

Week 8: 试运营
├── 体验课邀约（周边学校家长、社区引流）
│   └── 输出: 体验课报名表
├── 试运行（1-2轮体验课、流程验证、问题收集）
│   └── 输出: 试运营问题清单、改进措施
└── 开业准备（宣传物料、开业活动方案、应急预案）
    └── 输出: 开业检查清单（见下方）
```

## 二、选址标准

### 2.1 基本条件

| 指标 | 要求 | 优先级 |
|------|------|--------|
| 面积 | >= 80 平方米（教学区 >= 50 平方米） | 必须 |
| 层高 | >= 3 米（机器人活动需要空间） | 必须 |
| 电力 | 独立回路，负载 >= 10kW（15-20台设备同时充电） | 必须 |
| 网络 | 支持 100Mbps+ 宽带接入 | 必须 |
| 消防 | 双安全出口，通过消防审批 | 必须 |
| 停车 | 门口可临时停车（家长接送） | 推荐 |

### 2.2 选址优先级

| 等级 | 位置类型 | 说明 |
|------|----------|------|
| A | 中小学周边 500 米内 | 步行可达，家长接送方便 |
| B | 社区商业中心 | 人口密集，品牌曝光度高 |
| C | 教育综合体 | 教育消费聚集，但竞争激烈 |
| D | 其他 | 需评估交通便捷性和目标客群密度 |

### 2.3 场地功能分区

```
教学场地布局（80 平方米参考方案）:
├── 教学区（50 平方米）
│   ├── 学生操作台 x 6-8 组（每组 2-3 人）
│   ├── 教师演示区（投影/大屏）
│   └── 设备充电柜
├── 体验区（10 平方米）
│   └── OTTO 机器人展示台（吸引来访家长）
├── 家长休息区（10 平方米）
│   └── 沙发、饮水机、宣传资料架
└── 设备存放区（10 平方米）
    ├── 备用设备存放（按 R33: 每15台配3台备用）
    ├── 3D 打印机（外壳制作）
    └── 维修工具区
```

## 三、装修与设备采购清单

### 3.1 装修清单

| 项目 | 规格/要求 | 预算参考 |
|------|-----------|----------|
| 地面 | 防滑地胶或木地板（机器人行走需要平整表面） | 80-120 元/平方米 |
| 墙面 | 浅色调涂料 + OTTO 品牌元素展示区 | 50-80 元/平方米 |
| 照明 | 每组操作台独立 LED 灯（色温 4000-5000K） | 3,000-5,000 元 |
| 弱电 | 每组操作台配备 4 口插座 + USB 充电口 | 5,000-8,000 元 |
| 网络布线 | Cat6 网线到每个操作组 + AP 安装点位 | 3,000-5,000 元 |
| 安全设施 | 灭火器 x 4、应急灯 x 2、安全出口标识 x 2 | 2,000-3,000 元 |
| 门头 | OTTO 品牌 logo + 店名 + 营业时间 | 8,000-15,000 元 |

## 四、人员招聘计划

### 4.1 岗位配置（单店）

| 岗位 | 人数 | 到岗时间 | 职责 |
|------|------|----------|------|
| 店长/运营负责人 | 1 | Week 1 | 全面运营管理、招生、家长沟通 |
| 教学老师 | 2-3 | Week 6-7 | 课程教学、设备维护、教学研发 |
| 前台/教务 | 1 | Week 7 | 学员管理、排课、家长接待 |
| 市场推广（兼职） | 1 | Week 7 | 地推、社群运营、活动策划 |

### 4.2 教学老师招聘要求

| 要求项 | 标准 |
|--------|------|
| 学历 | 大专及以上，理工科/教育类专业优先 |
| 年龄 | 22-35 岁 |
| 技能 | 基本电脑操作能力（浏览器使用、Office 办公） |
| 经验 | 有教育/培训行业经验优先，无经验可培训 |
| 特质 | 有耐心、善于与青少年沟通、动手能力强 |
| 加分项 | 了解编程/机器人/电子制作，有创客教育经验 |

### 4.3 运营人员招聘要求

| 要求项 | 标准 |
|--------|------|
| 学历 | 大专及以上 |
| 经验 | 1 年以上教育/培训行业前台或教务经验 |
| 技能 | 熟练使用微信群运营、基础文案撰写 |
| 特质 | 亲和力强、服务意识好、做事细心 |

## 五、资质办理清单

```
资质办理流程:
├── 营业执照（市场监管局）
│   ├── 经营范围: 教育咨询、科技培训、文化艺术交流
│   ├── 办理周期: 5-15 个工作日
│   └── 所需材料: 租赁合同、法人身份证、场地平面图
├── 消防安全检查（消防部门）
│   ├── 场地面积 >= 200 平方米需消防审批
│   ├── 办理周期: 10-20 个工作日
│   └── 所需材料: 场地平面图、消防设施清单、装修材料证明
├── 食品经营许可（如提供零食饮料）
│   ├── 非必须项，根据业务需求决定
│   └── 办理周期: 10-15 个工作日
└── 其他（视当地政策）
    ├── 文化艺术类培训备案
    ├── 环保审批（如有 3D 打印，关注通风要求）
    └── 广告审批（门头/户外广告）
```

## 六、试运营方案

### 6.1 试运营目标

| 目标 | 指标 |
|------|------|
| 体验课场次 | >= 4 场（每场 6-10 人） |
| 体验课转化率 | >= 25%（体验后报名正式课程） |
| 流程验证 | 课前-课中-课后全流程跑通 |
| 问题收集 | 形成试运营问题清单并解决 |

### 6.2 体验课流程

```
体验课流程（60 分钟）:
├── 签到接待（10 分钟）
│   ├── 家长签到、发放体验资料
│   ├── 学生贴姓名贴、分配操作组
│   └── 家长引导至休息区
├── 课程体验（35 分钟）
│   ├── OTTO 机器人介绍与演示（5 分钟）
│   ├── 学生动手操作：语音互动体验（10 分钟）
│   ├── 学生动手操作：图形化编程初体验（15 分钟）
│   └── 作品展示与互动（5 分钟）
├── 课程咨询（15 分钟）
│   ├── 课程体系介绍
│   ├── 收费方案说明
│   └── 报名优惠活动
└── 课后跟进（持续）
    ├── 24 小时内发送体验课照片与反馈
    ├── 3 天内电话回访家长意向
    └── 7 天内完成转化或标记为待跟进
```

## 七、开业前检查清单

### 7.1 场地与设施

- [ ] 租赁合同已签订，押金和首期租金已支付
- [ ] 消防审批通过（如需要）
- [ ] 装修完工并验收，无安全隐患
- [ ] 供电系统正常，负载测试通过
- [ ] 安全出口标识清晰，灭火器到位
- [ ] 应急照明正常工作
- [ ] 门头/招牌安装完成

### 7.2 设备与网络

- [ ] OTTO 机器人套件全部到货并验收
- [ ] 每台机器人通电测试通过（唤醒、动作、语音）
- [ ] 备用设备（3台）到位并同步配置（R33）
- [ ] 紧急停止按钮功能测试通过（R30）
- [ ] 教学平板配置完成，浏览器兼容性测试通过（R6）
- [ ] 5GHz Wi-Fi AP 部署完成，信号覆盖测试通过（R4）
- [ ] 15-20 台设备同时连接压力测试通过
- [ ] 充电柜安装完成，充电功能正常
- [ ] 投影仪/大屏安装调试完成

### 7.3 系统与账号

- [ ] 管理员账号注册完成
- [ ] 教师账号创建并分配权限（R24）
- [ ] 设备录入后台，状态监控正常（R31）
- [ ] 课程内容推送测试通过（R25）
- [ ] 安全过滤配置完成（R26）
- [ ] 排课系统配置完成
