# 贡献指南

感谢您对 OTTO 123 项目的关注！我们欢迎各种形式的贡献。

## 如何贡献

### 报告问题

如果您发现了问题，请通过 GitHub Issues 报告：

1. 检查是否已有相同问题
2. 使用问题模板创建新 Issue
3. 提供详细的复现步骤
4. 附上相关截图或日志

### 提交代码

1. Fork 本仓库
2. 创建您的功能分支 (`git checkout -b feature/AmazingFeature`)
3. 提交您的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 打开 Pull Request

### 贡献内容

- **课程资源**: 教案、PPT、视频教程
- **文档改进**: 修正错误、完善说明
- **代码贡献**: 固件优化、工具开发
- **翻译工作**: 多语言支持
- **经验分享**: 教学案例、使用心得

## 开发规范

### 文档规范

- 使用 Markdown 格式
- 文件名使用英文小写，单词间用连字符分隔
- 图片放在 `assets/images/` 目录
- 保持文档结构清晰

### 代码规范

- 遵循现有代码风格
- 添加必要的注释
- 提交前测试通过
- 保持提交信息清晰

## 社区行为准则

- 尊重他人，友善交流
- 接受建设性批评
- 关注社区最佳利益
- 禁止骚扰、歧视等行为
