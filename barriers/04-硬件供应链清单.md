## 三、供应链风险评估

### 3.1 风险矩阵

| 风险项 | 发生概率 | 影响程度 | 风险等级 | 应对策略 |
|-------|---------|---------|---------|---------|
| ESP32-S3 芯片缺货 | 中 | 高 | 🟠 中高 | 备选乐鑫ESP32-C3（功能相近） |
| 乐鑫涨价 | 低 | 中 | 🟡 中 | 提前锁货60-90天库存 |
| 3D打印交期延误 | 低 | 中 | 🟡 中 | 备选2家供应商 |
| 锂电池运输限制 | 低 | 高 | 🟡 中 | 分批采购、第三方仓库周转 |
| 单一供应商依赖 | 中 | 中 | 🟡 中 | 每个关键件找2个备选供应商 |
| 量产PCB质量不稳定 | 中 | 高 | 🟠 中高 | 先打样10片验证，再批量 |

| 器件 | 首选 | 替代1 | 替代2 | 替代风险 |

## 四、采购执行流程

### 4.1 MVP阶段采购流程

```
1. 需求确认 → 2. BOM核对 → 3. 询价比价 → 4. 下单付款 → 5. 验货入库 → 6. 组装测试
```

| 步骤 | 操作 | 负责人 | 时长 |
|-----|-----|-------|-----|
| 需求确认 | 确认机器人数+配置 | canmou | 即时 |
| BOM核对 | 核对物料清单+规格 | dev | 1天 |
| 询价比价 | 嘉立创+华秋+淘宝比价 | dev | 1天 |
| 下单付款 | 线上付款 | dev | 即时 |
| 验货入库 | 外观+功能抽检 | dev | 1天 |
| 组装测试 | 烧录固件+功能测试 | dev | 1-2天 |
| **合计** | | | **5-6天** |

### 4.2 验货标准

| 检查项 | 合格标准 | 不合格处理 |
|-------|---------|-----------|
| PCB外观 | 无毛刺、无短路痕迹 | 退换 |
| 芯片焊接 | 无虚焊、无偏移 | 退换 |
| 模组装 | 芯片型号与BOM一致 | 退换 |
| 舵机转动 | 360度顺畅无卡顿 | 退换 |
| Wi-Fi连接 | 能稳定连接AP | 退换 |
| 电池续航 | ≥2小时连续运行 | 退换 |
