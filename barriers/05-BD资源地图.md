| 类型 | 特征 | 切入优势 | 优先级 |

## 五、BD切入路径

### 5.1 晚托班合作流程

```
Step 1：找到决策人
├── 方式：实地走访/朋友推荐
├── 目标：见到老板/负责人（不是店员）
└── 验证问题："你们现在课后服务有什么特色？招生压力大吗？"

Step 2：激发兴趣
├── 痛点切入：差异化竞争、招生亮点
├── 展示：用手机演示 OTTO 语音对话（2分钟）
└── 话术："你们现在能拿出一个让孩子回家主动说的特色课吗？"

Step 3：给试用方案
├── 免费体验课：1节（90分钟），不收费
├── 提供：设备+教师+课程
└── 条件：体验课后给反馈，协助宣传

Step 4：成交
├── 定价：¥300-500/月/人
├── 周期：学期制（4个月）
└── 决策期：体验课后1周内
```

### 5.2 学校合作流程

```
Step 1：找到入口人
├── 方式：科技节参展/教师推荐/教育局关系
├── 目标：找到校内支持者（教师/科技辅导员）
└UT 验证问题："学校现在有哪些科技类课程？"

Step 2：进校体验
├── 形式：科技社团体验课（免费）
├── 提供：设备+教师+课程
└── 条件：在科技节/开放日展示

Step 3：打动决策者
├── 方式：学生作品展示 + 家长/校长观摩
├── 关键：让学生给校长"讲解"自己的作品
└UT 话术："孩子的热情是最好的证明"

Step 4：签约
├── 定价：¥1,280-1,580/学期（16课时）
├── 周期：学期制
└UT 决策期：1-2个月
```

## 六、话术模板

### 6.1 晚托班开场话术（电话/实地）

```
"您好，我叫XX，是做机器人创客教育的。我们有一套专门为晚托班设计的机器人课程，
孩子可以自己组装、编程、对话。

我注意到你们这边周边有几个小区，双职工家庭应该挺多的。
我想问一下，你们现在课后服务有没有什么特色课？
如果没有的话，我想约个时间带几台设备过来，给孩子们上一节体验课，不收费用。
您看这周方便吗？"
```

### 6.2 学校开场话术（教育局/校长）

```
"您好，我是做机器人创客教育的XX。我们的产品刚完成了内部验证，
现在想在杭州找2-3所学校合作试点。

我们的核心差异是：杭州唯一一款带AI语音对话的开源机器人，
学生可以自己组装、编程，参加竞赛。
整套方案比乐高便宜60%，但功能更多。

我想找机会给科技辅导员或教务主任演示一下，
最快下周就可以进校给孩子们上一节体验课。
您看怎么安排比较方便？"
```

### 6.3 微信发消息模板（朋友推荐场景）

```
"XX老师好！我是XX的朋友，推荐说您这边在找特色课项目。
我这边有一套机器人创客课程，专门针对课后服务场景的。

核心卖点：
1. 学生能自己组装机器人（动手体验）
2. 带AI对话功能（市场独家）
3. 比乐高便宜很多，但课程体系更完整
4. 可以参加白名单竞赛（升学加分）

我想约个时间带设备过来，给孩子们上一节免费体验课，
您看这周或下周方便吗？"
```

## 十、注意事项

| ⚠️ BD大忌 | ✅ 正确做法 |
|----------|----------|
| 一上来就谈价格 | 先了解对方需求和痛点 |
| 说"我们比乐高便宜" | 说"我们的差异化是什么，解决什么问题" |
| 承诺无法交付的东西 | 说得保守，做超预期 |
| 一次性联系后不跟进 | 3天内必须再次触达 |
| 忽视校内支持者 | 先搞定老师，再攻校长 |
